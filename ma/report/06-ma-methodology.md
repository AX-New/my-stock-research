# A股 MA 均线择时方法论

> 基于上证指数/深证成指/创业板指 + 30个申万一级行业验证
> 数据: 1991~2026（33年，21个牛熊周期）
> 日线 12000+ 信号 / 周线 1700+ 信号 / 月线 800+ 信号
> 本文档为操作指南，不是数据报告。详细数据见 `01~04` 系列报告

---

## 一、核心认知（所有策略的前提）

### 1.1 什么信号不能用

**MA金叉死叉在全量统计中接近抛硬币，不能单独作为交易依据。**

| 信号 | 日线数量(33年) | T+60胜率 | 判断 |
|------|-------------|---------|------|
| MA5/10金叉 | 422 | 52.5% | 噪音 |
| MA5/10死叉 | 423 | 49.6% | 噪音 |
| MA10/20金叉 | 203 | 52.3% | 噪音 |
| MA10/20死叉 | 202 | 49.2% | 噪音 |
| MA5/20金叉 | 238 | 50.0% | 噪音 |
| MA20/30金叉 | 141 | 52.5% | 噪音 |

所有MA交叉的全量胜率在 45~55%，与随机无异。**"金叉就买、死叉就卖"是最常见的散户亏损模式之一。**

**假突破信号同样不可靠：**

| 信号 | 数量 | T+60胜率 | 判断 |
|------|------|---------|------|
| MA20假跌破(回升) | 196 | 50.5% | 噪音 |
| MA20假突破(回落) | 196 | 45.6% | 噪音 |
| MA60假跌破(回升) | 87 | 52.9% | 噪音 |

**方向突破在很多行业胜率不足40%：**

Layer 3行业验证发现，MA20方向突破在有色金属(25%)、食品饮料(27%)、计算机(32%)、汽车(33%)等行业的胜率远低于随机。MA60方向突破在房地产(22%)、交通运输(25%)、纺织服饰(25%)同样失效。**不能把"突破均线就买入"当作通用规则。**

### 1.2 什么决定信号有效性

**牛熊背景是第一判断维度。** 这是MA研究与MACD研究完全一致的最重要结论：

| 信号 | 牛市T+60 | 牛市胜率 | 熊市T+60 | 熊市胜率 |
|------|---------|---------|---------|---------|
| MA60乖离率超卖(买) | +19.82% | 94.7% | -4.16% | 38.2% |
| MA20乖离率超卖(买) | +17.52% | 83.2% | -3.45% | 36.1% |
| MA20方向突破(买) | +10.06% | 75.7% | -4.75% | 39.2% |
| MA20支撑(买) | +6.65% | 72.8% | -2.53% | 36.4% |
| MA60乖离率过热(卖) | +8.16% | 41.1% | -5.36% | 74.8% |
| MA60方向跌破(卖) | +9.02% | 20.0% | -5.79% | 70.4% |
| MA20阻力(卖) | +6.47% | 32.2% | -3.74% | 69.3% |

**同一个信号，牛市胜率70~95%，熊市胜率25~40%。** 不判断牛熊就做交易 = 赌博。

### 1.3 均线周期的选择

不是所有均线都平等。数据清晰表明：

- **MA20和MA60远优于MA5和MA10。** MA60乖离率超卖在牛市中胜率94.7%，MA20超卖83.2%，而MA5超卖仅75.7%、MA10超卖80.7%
- **周线是最稳定的操作频率。** 周线MA20乖离率超卖牛市胜率96.4%（28次），高于日线83.2%（279次），且信号频率适中
- **月线看方向，周线做操作，日线精确入场。** 这与MACD的多周期框架一致

### 1.4 MA vs MACD：各自的角色

两个指标各有擅长，不是替代关系：

| 维度 | MACD更强 | MA更强 |
|------|---------|--------|
| 动能拐点 | DIF极值反转率85~100% | 无对应信号 |
| 趋势确认 | 零轴位置 | 均线排列 + 支撑阻力 |
| 超买超卖 | DIF极值 | **乖离率极值（MA最强信号）** |
| 背离检测 | 100%短期胜率 | 无对应信号 |
| 支撑/阻力位 | 无 | **MA20/60支撑阻力** |
| 信号频率 | 较低（极值年均3~4次） | 较高（乖离率更频繁） |

**核心定位：MACD判动能拐点，MA判趋势位置和乖离回归。两者共振时信号最强。**

## 二、信号分级（按实战价值排序）

### 信号 A级：乖离率极值（最高优先级）

**原理：** 价格偏离均线过远时，有均值回归的内在力量。乖离率(bias)到达历史极值分位，意味着短期价格已过度偏离。

**这是MA体系中最有价值的信号，类比MACD的DIF极值。**

**可靠性数据（上证指数，牛市环境）：**

| 信号 | 日线T+60 | 周线T+13 | 月线T+12 |
|------|---------|---------|---------|
| MA60 bias极低(买) | 94.7% / +19.82% | 87.5% / +9.16% | 54.9% / +7.49% |
| MA20 bias极低(买) | 83.2% / +17.52% | 96.4% / +11.61% | 67.5% / +10.21% |
| MA10 bias极低(买) | 80.7% / +14.87% | 97.1% / +8.64% | 46.2% / +1.38% |
| MA60 bias极高(卖) | 74.8% / -5.36% | 76.6% / -4.97% | 44.4% / +26.03%* |
| MA20 bias极高(卖) | 67.9% / -3.97% | 70.6% / -5.51% | 32.1% / +27.16%* |

> *月线乖离率过热的"胜率"低是因为评估窗口(1年)太长，牛市中长期看涨覆盖了短期回调。短期(T+1月)月线bias过热的卖出胜率达到61~69%。

**关键特征：**
- **周线乖离率超卖是最稳定的买入信号**：牛市胜率96~97%，这是MA体系中最接近"确定性"的信号
- **日线MA60 bias极低的牛市胜率高达94.7%**，是所有日线信号中最高的
- 买入信号（bias极低）整体强于卖出信号（bias极高），这与A股"急跌慢涨"的特性一致
- 乖离率信号在全量（不区分牛熊）统计中胜率仍有51~53%，虽然不高，但结合牛熊判断后爆发为70~97%

**三大指数验证：** 日线MA20 bias超卖牛市胜率：上证83.2%、深证80.8%、创业板80.4%，极差仅2.8%，高度一致。

**操作要点：**
- 乖离率触发时不要急于"精确抄底"，等1~2根K线确认回归趋势
- 周线级别最稳定，日线次之，月线样本太少需谨慎
- 牛市中bias极低 = 加仓机会；熊市中bias极低 = 仍有36~38%概率继续跌，不可重仓

### 信号 B级：均线支撑/阻力 + 方向突破（条件性使用）

**原理：** 均线本身构成动态支撑阻力位。价格回踩均线获支撑 = 趋势延续；价格反弹受阻 = 趋势压制。方向突破则代表均线方向发生转变。

**仅在已确认牛市或熊市中使用。全量统计胜率勉强过半，加上牛熊筛选后才有价值。**

**牛市中的买入信号：**

| 信号 | 牛市T+60胜率 | 牛市均值 | 日线样本 |
|------|-----------|---------|---------|
| MA20方向突破 | 75.7% | +10.06% | 107 |
| MA20支撑 | 72.8% | +6.65% | 274 |
| MA60支撑 | 71.7% | +6.00% | 110 |
| MA10/20金叉 | 72.7% | +8.85% | 92 |

**熊市中的卖出信号：**

| 信号 | 熊市T+60胜率 | 熊市均值 | 日线样本 |
|------|-----------|---------|---------|
| MA60方向跌破 | 70.4% | -5.79% | 71 |
| MA20阻力 | 69.3% | -3.74% | 274 |
| MA60阻力 | 69.6% | -3.78% | 138 |
| MA20方向跌破 | 63.4% | -3.50% | 145 |

**周线支撑信号特别突出：** 周线MA20支撑在牛市中胜率85.0%（21次），短期(T+2周)胜率更是高达85.4%（41次全量），是仅次于乖离率极值的买入信号。

**行业例外清单（方向突破不可用的行业）：**
- MA20方向突破失效: 有色金属(25%)、食品饮料(27%)、计算机(32%)、汽车(33%)、房地产(34%)
- MA60方向突破失效: 房地产(22%)、交通运输(25%)、纺织服饰(25%)、通信(27%)、建筑装饰(27%)

**这些行业不要用"突破均线就买入"的策略，用乖离率极值替代。**

### 信号 C级：均线排列 + 粘合发散（趋势确认工具）

**不用于发出交易信号，用于确认已有判断。**

| 信号 | 全量T+60胜率 | 牛市胜率 | 熊市胜率 |
|------|-----------|---------|---------|
| 多头排列形成(买) | 54.0% | 63.3% | 34.6% |
| 空头排列形成(卖) | 51.8% | 12.5%* | 64.3% |
| 粘合后向下发散(卖) | 55.9% | 26.3% | 67.3% |
| 粘合后向上发散(买) | 47.5% | 59.4% | 33.3% |

> *空头排列在牛市中卖出胜率仅12.5%，意味着87.5%的情况后续是涨的 — 牛市中出现空头排列反而是加仓机会。

**正确用法：**
- 多头排列 + bias极低 = 强化买入信号（趋势向上 + 短期超卖）
- 空头排列 + bias极高（卖出方向）= 确认卖出（但独立使用不可靠）
- 粘合发散可作为波段起点的参考，但样本太少（粘合61次、发散68次），不宜作为独立依据

### 信号 D级：MA交叉（谨慎使用）

**只有在牛市中、且使用MA10/20以上的周期组合时，金叉才有参考价值。**

| 信号 | 牛市T+60胜率 | 牛市均值 | 熊市胜率 |
|------|-----------|---------|---------|
| MA10/20金叉 | 72.7% | +8.85% | 36.0% |
| MA5/20金叉 | 72.2% | +8.54% | 34.3% |
| MA10/30金叉 | 73.3% | +8.69% | 37.8% |
| MA20/30金叉 | 70.5% | +9.08% | 38.5% |
| MA5/10金叉 | 72.4% | +9.57% | 36.8% |

牛市中70%+的胜率看起来不错，但这些信号的价值完全来自牛市背景本身。**如果你已经判断了牛市，直接用A级（乖离率）和B级（支撑）信号就够了，金叉只是锦上添花。**

**月线金叉的特殊价值：** MA10/30金叉在月线上1年收益+27.37%（9次，77.8%胜率）、MA20/30金叉1年收益+26.91%（7次，66.7%胜率）。样本极小，但暗示月线长周期金叉可能标志着大级别趋势启动。仅作为观察指标，不单独使用。

### 噪音信号（不使用）

以下信号在任何市场环境下都不可靠，直接排除：

| 信号 | 原因 |
|------|------|
| MA5/10金叉死叉 | 全量胜率49~53%，噪音 |
| 所有假突破信号 | 全量胜率45~53%，定义本身就有模糊性 |
| 周线MA60方向突破 | 全量胜率仅26.3%，19次中只有5次上涨 |
| 周线粘合发散 | 样本仅6~8次，无统计意义 |
| 不区分牛熊的任何信号 | 全量胜率几乎全部在45~55%之间 |

## 三、多周期协同框架

### 3.1 三个周期各自的角色

| 周期 | 作用 | 核心MA信号 | 操作频率 |
|------|------|----------|---------|
| **月线** | 大方向 + 乖离率定位 | MA20 bias极值 + 方向突破 | 月度评估 |
| **周线** | 核心操作频率 | MA20 bias极值(96%+胜率) + MA20支撑(85%胜率) | 每周评估 |
| **日线** | 精确择时（仅辅助） | MA60 bias极值 + MA20支撑/阻力 | 每日监控 |

### 3.2 与MACD的协同

MA和MACD的信号在不同维度上互补，共振时信心最高：

**底部共振（最强买入信号）：**
```
月线 MACD DIF极小值 + 月线 MA20 bias极低 → 大级别底部确认
周线 MACD DIF极小值 + 周线 MA20 bias极低 → 波段底部确认
日线 价格回踩MA20获支撑 + 日线 MACD DIF开始回升 → 精确入场点
```

**顶部共振（最强卖出信号）：**
```
月线 MACD DIF极强多头 + 月线 MA60 bias极高 → 大级别见顶预警
周线 MACD DIF极大值 + 周线 MA60 bias过热 → 波段顶部确认
日线 价格反弹受阻MA20 + 日线 MACD DIF开始回落 → 精确出场点
```

**信号矛盾时的处理：**
- MACD看跌 + MA支撑住 → 等待，不急于操作，观察支撑是否有效
- MACD看涨 + MA阻力位 → 轻仓试探，突破阻力后加仓
- 以大周期为准：月线信号 > 周线信号 > 日线信号

### 3.3 信号权重

```
乖离率极值 > 支撑阻力 > 方向突破 > 均线排列 > MA交叉
月线信号 > 周线信号 > 日线信号
MACD + MA 共振 > 单指标信号
```

## 四、具体操作策略（3个模板）

### 策略1：周线乖离率均值回归（核心策略）

**目标：** 利用周线乖离率极值做波段操作，这是MA体系中胜率最高的策略。

**触发条件：**
- 周线MA20 bias 低于历史分位Q10（极度超卖）
- 已确认处于牛市背景（参考MACD月线零轴位置 + 牛熊判断）
- 至少等1根周线确认bias开始回升

**操作：**
- 在月线仓位框架内加仓
- 加仓幅度：仓位的20~30%

**预期收益：**
- 周线T+13(1季度)：96.4%胜率，平均+11.61%（28次）
- 这是MA体系中最接近"确定性"的交易机会

**止损：**
- bias继续创新低且MACD未出现底部信号 → 减仓观望
- 如果连续2周bias继续恶化，说明牛熊判断可能有误，降至最低仓位

**对称操作（卖出方向）：**
- 周线MA60 bias 高于Q90 + 熊市背景 → 减仓
- 熊市中周线MA60 bias过热胜率76.6%

### 策略2：均线支撑波段（趋势中的加仓策略）

**目标：** 在多头趋势中，利用价格回踩均线的支撑做加仓。

**触发条件：**
- 价格回踩MA20（日线）或MA60，距离均线 < 1%
- 处于多头排列中（MA5 > MA10 > MA20 > MA60）
- 次日收盘价站回均线上方（确认支撑有效）
- 已确认牛市背景

**操作：**
- 加仓，幅度：仓位的10~20%
- 入场点：确认支撑有效的次日

**预期收益：**
- 日线MA20支撑牛市T+60：72.8%胜率，平均+6.65%（274次）
- 周线MA20支撑牛市T+2周：85.0%胜率（21次）

**止损：**
- 收盘价跌破支撑均线超过2%且未回升 → 支撑失败，减仓
- 均线排列从多头转为交叉 → 趋势可能转变，降低仓位

**注意：** 此策略在以下行业效果差，应改用策略1（乖离率）：
- 建筑装饰（MA20支撑胜率39.8%）
- 交通运输（41.8%）
- 计算机（41.8%）

### 策略3：熊市防御（减仓/空仓策略）

**目标：** 在熊市中利用MA信号确认下跌趋势，避免抄底陷阱。

**核心规则：熊市中所有买入信号的胜率 < 40%，不要逆势。**

| 熊市中的买入信号 | T+60胜率 | 结论 |
|---------------|---------|------|
| MA60 bias极低 | 38.2% | 不买 |
| MA20 bias极低 | 36.1% | 不买 |
| MA20支撑 | 36.4% | 不买 |
| MA20方向突破 | 39.2% | 不买 |
| 所有金叉 | 34~38% | 不买 |

**触发减仓的卖出信号（熊市中）：**

| 信号 | 熊市T+60胜率 | 操作 |
|------|-----------|------|
| MA60 bias过热 | 74.8% | 减仓30~50% |
| MA60方向跌破 | 70.4% | 减仓30~50% |
| MA20阻力 | 69.3% | 不加仓，维持低仓位 |
| MA60阻力 | 69.6% | 不加仓，维持低仓位 |
| 粘合后向下发散 | 67.3% | 确认下跌趋势 |

**熊市中的正确做法：**
1. 仓位控制在30%以下
2. 只在月线MACD DIF极强空头时才考虑逆向建仓（参考MACD方法论策略1）
3. 所有MA买入信号在熊市中都是噪音，无论看起来多"像底部"

## 五、行业差异

### 5.1 类别策略差异

Layer 3验证了30个申万行业的MA信号表现，按5大类别呈现不同特征：

| 类别 | 乖离率超卖买入 | 方向突破买入 | 推荐策略 |
|------|-------------|-----------|---------|
| **大消费** | 最佳(55.5%) | 差(46.8%) | 乖离率为主，不用方向突破 |
| **周期** | 中等(54.9%) | 差(39.6%) | 乖离率 + 方向跌破(卖出60.6%) |
| **科技成长** | 中等(50.2%) | 中等(45.7%) | 乖离率为主，MA60过热卖出强(61.4%) |
| **稳定制造** | 中等(51.9%) | 差(41.7%) | 乖离率为主，多头排列辅助(52.1%) |
| **金融地产** | 最弱(48.6%) | 差(43.4%) | 信号整体偏弱，以MACD为主导 |

**关键发现：**
- 跨类别差异仅~7个百分点（bias信号），MA信号比MACD的行业差异更小，具有更好的普适性
- 方向突破信号在所有类别都偏弱（39~47%），不推荐作为独立交易依据
- 周期类的方向跌破卖出信号最可靠（60.6%），可以单独使用

### 5.2 行业例外清单

**以下行业显著偏离通用结论，需要特殊处理：**

**乖离率超卖也不好用的行业：**
- 建筑装饰：MA20 bias超卖胜率39.8%、MA60 bias超卖38.2%（唯一连乖离率都失效的行业）
- 传媒：MA60 bias超卖胜率39.4%
- 钢铁：MA60 bias超卖胜率41.5%

**方向突破全面失效的行业（Top 5）：**
- 有色金属(25%)、食品饮料(27%)、计算机(32%)、汽车(33%)、房地产(34%)

**乖离率信号最强的行业（Top 3）：**
- 食品饮料：MA20 bias超卖75.3%、MA60 bias超卖72.8%
- 国防军工：MA20 bias超卖71.8%
- 农林牧渔：MA20 bias超卖67.9%

> 食品饮料乖离率信号最强，但方向突破最弱（27%），说明消费股的价格倾向于围绕均线波动（适合均值回归），但均线突破本身没有趋势延续的含义。这与MACD方法论中发现的大消费"趋势延续"特性形成互补视角。

## 六、风控规则

### 6.1 绝不做的事

1. **不根据单一MA金叉/死叉交易** — 全量胜率≈50%，是噪音
2. **不在熊市中用MA支撑位抄底** — 熊市MA20支撑胜率仅36.4%，支撑会不断被击穿
3. **不在熊市中把bias极低当买入信号** — 熊市中MA60 bias极低胜率仅38.2%，"便宜"可以更便宜
4. **不用MA60方向突破做周线级别操作** — 周线全量胜率仅26.3%，是所有信号中最差的
5. **不在建筑装饰行业使用任何MA信号** — 唯一连乖离率都失效的行业

### 6.2 止损规则

- **乖离率策略止损：** bias触发后连续2个周期继续恶化（创新极值），说明不是正常均值回归而是趋势加速，止损出场
- **支撑策略止损：** 收盘价跌破支撑均线超过2%且次日未收回，支撑失效
- **总仓位止损：** 当MA和MACD信号持续矛盾超过2个月时，降至最低仓位重新评估
- **黑天鹅保护：** 单日跌幅>5%，暂停所有策略等待信号重新明确

### 6.3 信号失效场景

- **政策突变（熔断、新冠等）：** 乖离率极值可能被连续击穿，支撑阻力完全失效
- **牛熊转换期：** 牛转熊的前1~2个月，所有买入信号的胜率从70%+骤降至40%以下，此时最易亏损
- **超长横盘市（2012~2014年）：** 均线反复粘合交叉，所有信号大量触发但无趋势可言，手续费侵蚀
- **流动性枯竭：** 极端缩量行情下，均线支撑/阻力的意义减弱

## 七、方法论的局限性

1. **牛熊判断本身是难点：** 策略假设能判断牛熊，但实时判断有滞后性。这是MA和MACD方法论共同的前提依赖
2. **样本量限制：** 月线信号样本极少（MA20方向突破仅14次），统计显著性有限；周线粘合发散仅6~8次，不具备统计意义
3. **行业验证充分，个股待验证：** 3大指数+30个行业已通过，但个股MA行为未验证
4. **乖离率阈值的时变性：** "极值"是基于历史分位数定义的，市场结构变化（如机构化、量化交易增多）可能导致阈值漂移
5. **方向突破信号整体偏弱：** 与技术分析教科书的经典结论不同，A股市场的MA突破信号胜率普遍不足50%，可能与A股散户占比高、假突破多有关
6. **未考虑交易成本和滑点：** 乖离率策略的操作频率较高，实际收益会被手续费侵蚀
7. **与MACD的协同未经统计验证：** 两个指标共振的量化效果尚未通过回测验证，目前仅为定性推断
