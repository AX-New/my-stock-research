# MA均线研究 - 脚本说明

## 计算脚本

| 脚本 | 用途 |
|------|------|
| `compute_index_ma.py` | 指数/申万行业MA均线全市场计算（7条SMA + 4个乖离率） |
| `compute_stock_ma_signals.py` | 个股MA信号事件回填，从原始K线现场计算MA并检测信号 |

## 分析脚本

| 脚本 | 用途 |
|------|------|
| `analyze_index_ma.py` | Layer 1: 指数MA均线信号全维度分析（信号统计 + 收益 + 牛熊交叉） |
| `analyze_sw_ma.py` | Layer 3: 申万一级行业MA信号分析，验证指数结论的行业适用性 |
| `compare_index_ma.py` | 三大指数MA信号对比分析，找出普适性结论和条件性结论 |

## 基础模块

| 脚本 | 用途 |
|------|------|
| `config.py` | 数据库连接配置（读 my_stock / 写 stock_ma） |
| `database.py` | 双引擎初始化（read_engine + write_engine） |
| `db_utils.py` | batch_upsert 批量写入工具 |
| `kline_loader.py` | K线数据加载，复用MACD模块的加载函数 |
| `models.py` | MA均线表模型定义（15张表：12张数据表 + 3张信号表） |
| `signal_detector_ma.py` | MA信号检测引擎（乖离率极值/方向突破/假突破/支撑阻力/排列/粘合/交叉） |
| `bull_bear_phases.py` | 牛熊周期数据，从MACD研究模块复用 |
