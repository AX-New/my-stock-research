# 研究方法论

## 策略定位

**价值投资为主体，技术指标做择时**，两者有机结合。

### 三层操作模式

**1. 长线持股（价值投资层）**
- 基于行业研究、长期趋势判断选股建仓
- 决策依据：基本面、行业前景，不是简单的技术指标
- 持股周期以年为单位

**2. 持股中波段套利（增强层）**
- 在长线持仓过程中，利用技术分析进行波段操作
- 目的：增厚利润 / 对冲错误决策产生的成本
- 参考周期：日线、月线、年线

**3. 纯技术套利（独立层）**
- 基于指标数据，纯技术面操作，打完就走
- 不依赖价值判断，纯粹的技术策略
- 参考周期：日线、月线、年线

## 分析原则

- **从顶底出发，反向研究指标特征**：不是看指标发出什么信号，而是先确认历史上的顶和底，再观察那些时刻各指标呈现什么特征
- 允许结合其他指标/数据进行共振分析，多信号验证提高可靠性
- 单一指标产生的信号需要其他维度数据佐证

```
传统思路（我们不用）：MACD金叉 → 预测上涨 → 验证准确率
我们的思路：        确认历史顶底 → 观察顶底时的指标特征 → 提炼共性规律
```

## 可共振的指标/数据

| 类别 | 指标 | 作用 |
|------|------|------|
| 趋势 | MACD | 趋势方向与动能变化 |
| 量价 | 成交量/换手率 | 验证趋势的真实性 |
| 均线 | MA/EMA | 支撑压力、趋势确认 |
| 强弱 | RSI | 超买超卖判断 |
| 波动 | 布林带 | 波动区间、突破信号 |
| 资金 | 北向资金/主力资金流 | 资金面验证 |
| 基本面 | 财报/估值 | 价值投资层的核心依据 |

## 时间周期 × 作用对象

| | 指数（大盘环境） | 板块（行业轮动） | 个股（买卖点） |
|--|----------------|----------------|--------------|
| 年线 | 牛熊大周期 | 行业长期兴衰 | 长线建仓/清仓 |
| 月线 | 中长期趋势 | 行业阶段性强弱 | 波段加减仓 |
| 周线 | 中期趋势确认 | 板块强弱切换 | 中线加减仓 |
| 日线 | 短期风险预警 | 短期资金流向 | 短线买卖点 |
