# 牛熊分析标准（强制约束）

## 核心规则

**每个信号的分析都必须按牛市/熊市分别统计，这是不可省略的核心步骤。**

MACD 研究最重要的结论：同一信号在牛熊中表现完全相反。例如金叉在牛市胜率89%，熊市仅33%。如果只看全量汇总（~50%），会错误判定为噪音。

## 执行标准

- 每个信号输出三行统计：**全量 / 牛市 / 熊市**
- 结论判定以牛熊分组为准，全量仅供参考

## 牛熊子类型（不是简单二元划分）

| 子类型 | 代表区间 | 特征 |
|--------|---------|------|
| **大牛市** | 998→6124, 1850→5178 | 趋势强劲，部分指标高位不回归 |
| **反弹牛** | 1665→3478 | 快速拉升，指标滞后严重 |
| **慢牛/结构牛** | 2638→3587, 2441→3732 | 价格贴均线走，支撑/阻力信号可能更有效 |
| **急跌熊** | 6124→1665, 5178→2638 | 支撑全部失效，信号反转极快 |
| **阴跌熊** | 3478→1850(46月), 3732→2635(36月) | 反复缠绕，假信号最多 |
| **中等熊** | 3587→2441 | 跌幅有限，部分信号仍有效 |

## 注意事项

- 某些信号可能只在特定子类型中有效（如均线支撑只在慢牛可靠，急跌熊中失效）
- 全量"牛市"统计可能掩盖大牛市与慢牛的差异，必要时进一步细分
- 早期数据(1990-2005)参考价值有限，核心样本是2005年后的6轮牛熊
- 创业板有独立行情，不能简单套用上证的牛熊划分
