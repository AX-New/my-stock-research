1. SSH 隧道已启动（`Desktop\Tssh-tunnel.bat`）

## 脚本列表

### config.py
**用途**：数据库连接配置和全局常量

关键配置：
| 变量 | 默认值 | 说明 |
|------|--------|------|
| `MY_STOCK_DB_URI` | port 3307 | my_stock 行情库 |

| `START_DATE` | 2025-03-15 | 默认回测起始 |
| `END_DATE` | 2026-03-19 | 默认回测结束 |

### data_loader.py
**用途**：加载策略所需数据

| 函数 | 说明 |
|------|------|
| `load_heat_data(start, end)` | 加载热度排名（popularity_rank） |
| `load_price_data(start, end)` | 加载前复权行情（market_daily + adj_factor） |
| `load_trading_days(start, end)` | 加载交易日历（trade_cal） |
| `load_index_data(start, end)` | 加载沪深300（index_daily） |
| `load_data_bundle(start, end)` | 一次性加载所有数据（推荐使用） |

```python
from heat.scripts.data_loader import load_data_bundle

bundle = load_data_bundle('2025-03-15', '2026-03-19')
# bundle = {'heat_df': ..., 'trading_days': ..., 'price_df': ..., 'index_df': ...}
```

### strategy.py
**用途**：HeatRotationStrategy 策略核心类

**默认参数**：
```python
DEFAULT_PARAMS = {
    'lookback': 20,           # 回看窗口（天）
    'buy_threshold': 0.80,    # 买入阈值
    'sell_threshold': 0.20,   # 卖出阈值
    'max_hold_days': 9999,    # 最大持仓天数
    'min_deal_amount': 5e7,   # 最低成交额（元）
    'n_positions': 1,         # 同时持仓数
    'sort_by': 'rank_surge',  # 排序字段
}
```

**使用示例**：
```python
from heat.scripts.strategy import HeatRotationStrategy
from heat.scripts.data_loader import load_data_bundle

bundle = load_data_bundle('2025-03-15', '2026-03-19')
strategy = HeatRotationStrategy()

# 用默认参数运行
result = strategy.run(bundle)
print(result['metrics'])

# 用自定义参数运行
result = strategy.run(bundle, params={'lookback': 30, 'buy_threshold': 0.85})
```

### optimizer.py
**用途**：HeatOptimizer 参数优化器（两阶段：网格搜索 + MLP代理模型）

**参数空间**（约 1500 组合，实际计算约 300 次）：
```python
DEFAULT_PARAM_SPACE = {
    'lookback':        [10, 15, 20, 30, 40],
    'buy_threshold':   [0.70, 0.75, 0.80, 0.85, 0.90],
    'sell_threshold':  [0.10, 0.15, 0.20, 0.25, 0.30],
    'min_deal_amount': [1e7, 3e7, 5e7, 1e8],
    'n_positions':     [1, 2, 3],
}
```

**使用示例**：
```python
from heat.scripts.optimizer import HeatOptimizer
from heat.scripts.strategy import HeatRotationStrategy
from heat.scripts.data_loader import load_data_bundle

bundle = load_data_bundle('2025-03-15', '2026-03-19')
strategy = HeatRotationStrategy()
optimizer = HeatOptimizer(strategy, bundle, metric='sharpe')
results = optimizer.run()
optimizer.save_results(results, 'heat/scripts')
```

### run_optimization.py
**用途**：参数优化 CLI 入口（推荐直接使用此脚本）

```bash
# 完整优化（Phase1 网格搜索 + Phase2 AI优化）
python heat/scripts/run_optimization.py \
    --start 2025-03-15 \
    --end 2026-03-19 \
    --output F:/projects/my-stock-research/backtest/output

# 仅网格搜索（跳过 AI 优化）
python heat/scripts/run_optimization.py --skip-ai

# 优化目标改为年化收益
python heat/scripts/run_optimization.py --metric annual_return --skip-ai

# 跳过 OrderBasedEngine 回测（仅优化参数）
python heat/scripts/run_optimization.py --skip-backtest
```

**输出文件**：
| 文件 | 位置 | 说明 |
|------|------|------|
| `best_params.json` | `heat/scripts/` | 最优参数（live_signal.py 读取） |
| `optimization_results.csv` | `heat/scripts/` | 所有优化结果 |
| `top20_results.csv` | `heat/scripts/` | Top 20 结果 |
| `best_params_orders.csv` | `backtest/output/` | OrderBasedEngine 订单格式 |

### live_signal.py
**用途**：每日实盘信号生成（收盘后 17:10 运行）

```bash
python heat/scripts/live_signal.py
```

**输入**：
- `best_params.json`（如不存在，使用默认参数）
- `state.json`（当前持仓状态，自动维护）
- my_trend.popularity_rank（最近 lookback+10 天数据）

**输出**：
- `signals/YYYYMMDD_signal.json`（当日信号）
- `state.json`（更新持仓状态）
- 控制台信号摘要

**信号格式示例**：
```json
{
  "date": "2026-03-20",
  "signals": [
    {"action": "sell", "stock_code": "000001", "reason": "heat_position(0.15) <= sell_threshold(0.20)"},
    {"action": "buy", "stock_code": "002521", "heat_position": 0.95, "rank_surge": 3.2}
  ]
}
```

## 运行顺序建议

```
Step 1：验证数据库连接
  mysql -h 127.0.0.1 -P 3310 -u root -proot my_trend -e "SELECT MAX(date) FROM popularity_rank;"
  mysql -h 127.0.0.1 -P 3307 -u root -proot my_stock -e "SELECT MAX(trade_date) FROM market_daily;"

Step 2：运行参数优化（一次性，约 10-20 分钟）
  python heat/scripts/run_optimization.py --start 2025-03-15 --end 2026-03-19

Step 3：每日实盘信号（收盘后 17:10）
  python heat/scripts/live_signal.py

Step 4：查看信号
  cat heat/scripts/signals/$(date +%Y%m%d)_signal.json
```

## 常见问题

**Q: 连接 my_trend 失败（port 3310）**
A: 检查 SSH 隧道是否启动（Desktop/Tssh-tunnel.bat）

**Q: ModuleNotFoundError: No module named 'heat'**
A: 确认在项目根目录运行（F:/projects/my-stock-research）

**Q: 优化时间过长**
A: 添加 `--skip-ai` 跳过 AI 优化，只做网格搜索（约 5-10 分钟）

**Q: scikit-learn 未安装**
A: `pip install scikit-learn`，或使用 `--skip-ai` 跳过 AI 优化
