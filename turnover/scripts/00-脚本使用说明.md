# 换手率研究 - 脚本说明

## 计算脚本

| 脚本 | 用途 |
|------|------|
| `compute_stock_turnover_signals.py` | 个股换手率信号事件回填，检测极端值/背离/区间/持续性/突变/交叉 |

## 分析脚本

| 脚本 | 用途 |
|------|------|
| `explore_turnover.py` | L1前置：上证指数换手率探索性分析（分布特征/牛熊统计/阈值确定） |
| `analyze_index_turnover.py` | Layer 1-2.5: 指数换手率信号全维度分析（信号统计 + 收益 + 牛熊交叉） |
| `analyze_industry_turnover.py` | Layer 3: 行业换手率信号分析，从个股按行业聚合市值加权平均换手率 |
| `analyze_stock_turnover.py` | Layer 4: 全市场个股换手率信号统计分析，汇总写入 stock_turnover_signal_stats |
| `analyze_time_stability_turnover.py` | Layer 3.5: 换手率信号时间稳定性验证，按4年分段检查跨时代和同环境稳定性 |

## 基础模块

| 脚本 | 用途 |
|------|------|
| `config.py` | 数据库连接配置（读 my_stock / 写 stock_turnover） |
| `database.py` | 双引擎初始化（read_engine + write_engine） |
| `db_utils.py` | batch_upsert 批量写入工具 |
| `models.py` | 换手率研究数据模型定义 |
| `signal_detector_turnover.py` | 换手率信号检测器（全向量化：极端值/背离/区间/持续性/突变/交叉） |
| `bull_bear_phases.py` | A股牛熊周期数据 + 标注工具 |
