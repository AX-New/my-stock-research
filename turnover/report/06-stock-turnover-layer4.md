# L4 全市场个股换手率信号验证报告

> 5346只个股 × 3频率 = 14221行统计 | 数据范围: 各股上市日~2026.03
> 统计基准: 信号数≥5 的股票才纳入（确保统计意义）
> 总信号量: 日线5346股 + 周线5282股 + 月线3593股

## 一、核心结论

**换手率信号从指数到个股发生了系统性的层级反转，这是换手率与MACD/RSI/MA最根本的区别。**

| 信号 | 指数级(L1-L2) | 个股级(L4) | 反转方向 |
|------|-------------|-----------|---------|
| **surge 暴增** | 68%看涨 | **52.5%**近随机 | ↓ 大幅衰减 |
| **extreme_high 超高** | 35%卖出失效 | **58.9%卖出有效** | 🔄 方向反转！ |
| extreme_low 超低 | 45%买入失效 | 51.4%近随机 | — 无变化 |
| price_down_vol_up | 54%弱买入 | 50.9%随机 | ↓ 衰减 |
| price_up_vol_down | 45%弱卖出 | 50.7%随机 | — 无变化 |
| zone_high | 45%卖出失效 | **55.1%弱卖出** | ↑ 回归有效 |
| zone_low | 54%弱买入 | 50.9%随机 | ↓ 衰减 |

**最重要的发现**：超高换手在指数级方向反转（35%卖出失效），但**在个股级回归为有效的卖出信号（58.9%）**。这说明：
- 指数超高换手 = 全市场资金入场 = 动能确认
- 个股超高换手 = 个股被过度炒作 = 主力出货

## 二、超高换手(extreme_high)——个股级最有价值的信号

### 2.1 全量统计

| 频率 | 有效股票数 | 平均卖出胜率 | 平均收益 |
|------|-----------|------------|---------|
| 日线 | 5270 | **58.9%** | -0.69% |
| 周线 | 5085 | **56.9%** | -0.12% |
| 月线 | 3497 | **62.1%** | -1.52% |

**三个频率全部有效，月线最强(62.1%)**。与指数级完全相反——个股超高换手是可靠的卖出信号。

### 2.2 胜率分布

| 卖出胜率区间 | 股票数 | 占比 |
|-------------|--------|------|
| 70%+ | 766 | 14.5% |
| 60-70% | 1637 | **31.1%** |
| 55-60% | 1019 | 19.3% |
| 50-55% | 833 | 15.8% |
| <50%（失效） | 1015 | 19.3% |

**45.6%的股票(2403只)超高换手卖出胜率≥60%**，14.5%的股票超过70%。仅19.3%失效。这是一个稳定、广泛有效的信号。

### 2.3 市值分层

| 市值 | 股票数 | 卖出胜率 | 平均收益 |
|------|--------|---------|---------|
| 微盘(<30亿) | 948 | **61.0%** | -1.16% |
| 小盘(30-100亿) | 2800 | **60.3%** | -1.25% |
| 中盘(100-300亿) | 1083 | 55.5% | +0.56% |
| 大盘(300-1000亿) | 332 | 54.3% | +0.61% |
| 超大盘(>1000亿) | 107 | 51.7% | +1.65% |

**完美的市值梯度**：越小盘超高换手卖出越有效（61%→52%）。原因：
- 小盘股超高换手 = 游资/散户炒作过度 = 回调概率高
- 大盘股超高换手 = 机构换仓/ETF调仓 = 不一定跌

### 2.4 换手率水平分层

| 换手率水平 | 股票数 | 卖出胜率 |
|-----------|--------|---------|
| 低换手(<2%) | 279 | 53.2% |
| 中换手(2-4%) | 1888 | 56.5% |
| 中高换手(4-7%) | 2214 | 59.7% |
| **高换手(>7%)** | 889 | **63.7%** |

**日常换手率越高的股票，超高换手卖出信号越强**。高换手股本身投机氛围浓，超过P95时意味着极度过热。

## 三、暴增信号(surge)——个股级大幅衰减

### 3.1 全量统计

| 频率 | 有效股票数 | 平均胜率 | 平均收益 |
|------|-----------|---------|---------|
| 日线 | 5346 | **52.5%** 上涨 | +1.34% |
| 周线 | 5282 | 54.3% 上涨 | +0.81% |
| 月线 | 3581 | **61.9%** 上涨 | **-1.93%** |

**日线暴增从指数级68%降至个股52.5%**，接近随机。原因：个股暴增的来源更杂——可能是利好公告、游资炒作、误操作等，不像指数暴增那样代表全市场情绪。

**月线暴增出现矛盾**：62%上涨胜率但平均收益-1.93%。说明大多数涨但少数暴跌拉低均值。月线暴增后的尾部风险极大。

### 3.2 胜率分布

| 上涨胜率区间 | 股票数 | 占比 |
|-------------|--------|------|
| 70%+ | 27 | 0.5% |
| 60-70% | 523 | 9.8% |
| 55-60% | 1190 | 22.3% |
| 50-55% | 1883 | **35.2%** |
| <50%（看跌） | 1723 | 32.2% |

绝大多数股票(57.5%)的暴增信号胜率在50-60%之间，仅0.5%超过70%。**个股暴增不是一个可依赖的买入信号。**

### 3.3 市值分层

| 市值 | 股票数 | 暴增胜率 |
|------|--------|---------|
| 微盘(<30亿) | 948 | 54.0% |
| 小盘(30-100亿) | 2800 | 53.2% |
| 中盘(100-300亿) | 1083 | 50.6% |
| 大盘(300-1000亿) | 332 | 49.5% |
| 超大盘(>1000亿) | 107 | 49.5% |

与指数级的"越小盘越强"一致，但个股级差距很小（54%→50%）。即使最强的微盘股也只有54%，不具备实战价值。

## 六、层级反转机制解析

为什么同一信号在指数和个股级方向相反？

```
超高换手(extreme_high)的层级反转逻辑:

指数层级:
  turnover_rate > P95 → 全市场交投活跃
  → 资金大规模入场 → 动能确认 → 后续涨 (35%卖出=65%涨)

个股层级:
  turnover_rate > P95 → 这只股票被过度关注
  → 游资/散户蜂拥 → 主力趁机出货 → 后续跌 (59%卖出有效)

暴增(surge)的层级衰减逻辑:

指数层级:
  当日/MA5 > 2倍 → 全市场情绪爆发
  → 系统性行情启动信号 → 后续涨 (68%胜率)

个股层级:
  当日/MA5 > 2倍 → 可能是:
  - 利好公告 (30%真利好, 70%已price-in)
  - 游资打板 (短期有效, T+20已退潮)
  - 大宗交易 (非情绪驱动)
  → 信号含义被稀释 → 胜率退化到52.5%
```
