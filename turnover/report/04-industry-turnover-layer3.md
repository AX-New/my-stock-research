# L3 申万行业换手率信号差异分析报告

> 31个申万一级行业 | 日线信号总量: 128,750 | 数据范围: 2013.11~2026.03
> 行业换手率 = 成分股市值加权换手率聚合，是"半聚合"层级指标

## 一、核心结论

**行业间换手率信号差异巨大，是所有四个指标（MACD/MA/RSI/换手率）中行业分化最严重的。**

- 暴增信号胜率极差：电力设备80%↑ vs 家用电器47%↓，差距33个百分点
- 超高换手方向反转不再普适：银行65%有效卖出，机械设备仅34%
- 行业特性（周期性/成长性/防御性）决定了换手率信号的含义完全不同

## 二、行业特征分类

### 2.1 暴增信号有效性与行业属性的关系

| 行业类型 | 代表行业 | 暴增胜率 | 原因 |
|---------|---------|---------|------|
| **高成长/科技** | 电子(74%)、计算机(71%)、电力设备(77%) | **70%+** | 暴增=资金追入赛道，动能强劲 |
| **中游制造** | 机械(75%)、汽车(71%)、轻工(74%) | **70%+** | 暴增=需求/订单预期升温 |
| **上游周期** | 有色(65%)、化工(68%) | **65%+** | 暴增=涨价预期确认 |
| **大盘蓝筹** | 银行(48%)、家电(47%)、石化(53%) | **<53%** | 暴增由机构/北向驱动，非情绪信号 |
| **政策驱动** | 房地产(49%)、环保(51%)、公用事业(49%) | **~50%** | 暴增=政策脉冲，不可持续 |
| **资源品** | 煤炭(50%) | **~50%** | 供给侧逻辑，暴增后常回落 |

**核心规律**：暴增信号在**成长性行业**最有效，在**蓝筹/防御性行业**失效。与L2.5的"越小盘越有效"结论一致——成长行业以中小盘居多，散户参与度高，换手率更能反映真实情绪。

## 三、超高换手(extreme_high)行业差异

| 类别 | 行业 | 卖出胜率 | 解读 |
|------|------|---------|------|
| **卖出有效** | 银行(65.2%)、非银金融(62.3%) | 金融行业超高换手=过度投机 |
| **卖出有效** | 国防军工(58.7%)、环保(58.2%) | 题材炒作后回归理性 |
| **中性** | 公用事业(54.9%)、农牧(51.4%)、传媒(50.3%) | 约50%，无信号价值 |
| **反转(买入)** | 机械设备(34.1%)、基础化工(35.5%) | 与指数级一致的方向反转 |
| **反转(买入)** | 交通运输(37.0%)、电力设备(38.2%) | 超高换手=动能确认 |

**行业维度的"方向反转"规律**：
- 金融/防御行业：超高换手**仍是卖出信号**（与传统分析一致）
- 成长/制造行业：超高换手**方向反转**为买入信号（与指数级结论一致）

这与暴增信号的行业分化逻辑完全吻合——成长行业的高换手代表资金涌入，金融行业的高换手代表过度投机。

## 六、实操决策树（行业维度）

```
行业换手率暴增
│
├─ 科技成长（电子/计算机/电力设备/通信）？
│   ├─ 牛市 → 强买入 ✅✅ (75-82%胜率)
│   └─ 熊市 → 仍可买入 ✅ (58-70%胜率，独立催化)
│
├─ 制造业（机械/汽车/轻工）？
│   ├─ 牛市 → 强买入 ✅✅ (75-78%胜率)
│   └─ 熊市 → 可谨慎买入 ✅ (57-71%胜率)
│
├─ 上游周期（有色/化工）？
│   ├─ 牛市 → 买入 ✅ (70-74%胜率)
│   └─ 熊市 → 忽略 (44-53%胜率)
│
├─ 金融（银行/非银/保险）？
│   └─ 任何环境 → 忽略 ❌ (48-58%胜率，机构驱动)
│
├─ 防御（公用事业/煤炭/石化）？
│   └─ 任何环境 → 忽略 ❌ (49-53%胜率)
│
└─ 顺周期（交运/建材/房地产）？
    ├─ 牛市 → 可买入 (61-65%胜率)
    └─ 熊市 → 坚决不买 ❌❌ (15-26%胜率，反弹陷阱!)
```

## 七、与前序报告的关系

| L1-L2.5结论 | L3验证结果 |
|-------------|-----------|
| 暴增=买入（指数级） | **部分确认**：成长行业确认，蓝筹/防御行业否定 |
| 超高换手方向反转 | **行业分化**：成长行业反转，金融行业传统有效 |
| 越小盘越强 | **逻辑一致**：成长行业≈中小盘，蓝筹行业≈大盘 |
| 量价背离需环境过滤 | **确认+细化**：行业也是重要过滤维度 |
