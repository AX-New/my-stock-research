# 资金流向研究 - 脚本说明

## 分析脚本

| 脚本 | 用途 |
|------|------|
| `analyze_stock_moneyflow.py` | 模块一：个股资金流分层验证（大单/小单/全口径/MOD修正因子分5档统计收益） |
| `analyze_industry_moneyflow.py` | 模块二：行业资金流双向验证（拥挤效应 + 资金流拐点预测力） |
| `analyze_moneyflow_resonance.py` | 模块三：聪明钱+拥挤度共振验证（大单流入 + 行业未过热 = 最优组合） |

## 基础模块

| 脚本 | 用途 |
|------|------|
| `config.py` | 数据库连接配置（读 my_stock / 写 stock_research） |
| `database.py` | 双引擎初始化（read_engine + write_engine） |
| `moneyflow_loader.py` | 资金流向数据加载模块（资金流/行情/基础信息） |
