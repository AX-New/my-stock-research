# ML 热度数据机器学习分析 - 脚本使用说明

## 概述

使用机器学习方法分析东财热度排名数据，挖掘热度特征与股票未来收益的关系。

**技术栈**: LightGBM + XGBoost + PyTorch LSTM (RTX 3080 GPU 加速)

## 脚本清单

| 脚本 | 用途 | 说明 |
|------|------|------|
| `config.py` | 配置 | 数据库连接、时间范围、ML 参数 |
| `database.py` | 数据库引擎 | my_stock (port 3307) + my_trend (port 3310) |
| `data_loader.py` | 数据加载 | 热度排名 + 前复权行情 + 沪深300指数 + 交易日历 |
| `feature_engine.py` | 特征工程 | 52个特征：热度排名/行情/大盘环境/交叉特征 |
| `models.py` | 模型封装 | LightGBM/XGBoost/LSTM 统一接口 |
| `run_heat_ml.py` | **主入口** | 完整 ML 流水线：数据→特征→训练→评估→分析 |

## 使用方法

### 前提条件

- SSH 隧道已启动（`Desktop\Tssh-tunnel.bat`）
- Python 环境已安装：scikit-learn, lightgbm, xgboost, torch, shap

### 运行命令

```bash
# 默认运行（LightGBM + XGBoost，预测5日收益率）
python -m ml.scripts.run_heat_ml

# 指定模型和目标
python -m ml.scripts.run_heat_ml --models lgb xgb --target 5 --task regression

# 包含 LSTM（需要 PyTorch + GPU）
python -m ml.scripts.run_heat_ml --models lgb xgb lstm --target 5

# 分类任务（预测涨跌方向）
python -m ml.scripts.run_heat_ml --task classification --target 5

# 不同预测周期
python -m ml.scripts.run_heat_ml --target 3   # 3日
python -m ml.scripts.run_heat_ml --target 10  # 10日
```

### 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `--models` | `lgb xgb` | 模型选择：lgb/xgb/lstm |
| `--target` | `5` | 预测未来 N 天 |
| `--task` | `regression` | 任务类型：regression/classification |

## 特征说明（52个）

### 热度排名特征（26个）

| 特征 | 说明 |
|------|------|
| `rank` | 原始排名 |
| `rank_pct` | 排名百分位（0=最热，1=最冷） |
| `rank_chg_1d/3d/5d/10d` | N日排名变化（正=变冷） |
| `rank_ma5/10/20` | 排名N日移动平均 |
| `rank_std5/10` | 排名波动率（N日标准差） |
| `heat_position_10/20/30` | 热度位置（0~1，1=近期最冷） |
| `rank_surge_10/20` | 排名弹性（当前/均值，>1=偏冷） |
| `days_in_top100/500` | 近20天进入TopN的天数 |
| `rank_min/max_20d` | 近20天最高/最低排名 |
| `change_rate` | 涨跌幅 |
| `volume_ratio` | 量比 |
| `turnover_rate` | 换手率 |
| `deal_amount` | 成交额 |
| `turnover_chg_1d` | 换手率日变化 |
| `volume_ratio_chg_1d` | 量比日变化 |

## 评估指标

### 回归任务

- **IC(Pearson/Spearman)**: 预测值与实际收益的相关性，>0.03 有效
- **ICIR**: IC的均值/标准差，衡量预测稳定性，>1.0 优秀
- **方向准确率**: 涨跌方向预测正确率
- **分组收益**: 按预测值分5组，看各组实际收益（多空收益 = Top - Bottom）
- **每日IC分析**: IC随时间的稳定性

### 分类任务

- **AUC**: ROC曲线下面积，>0.55 有效
- **准确率/精确率/召回率**: 标准分类指标
