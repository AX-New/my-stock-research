# 热度数据 ML 分析框架与初步发现
**创建时间**: 20260322 20:30

## 1. 研究目标

使用机器学习方法分析东财热度排名数据，挖掘热度特征与股票未来收益的关系。

**核心问题**：热度排名数据是否蕴含可预测股票收益的信息？哪些热度特征最具预测力？

## 2. 数据概况

| 数据源 | 范围 | 记录数 | 说明 |
|--------|------|--------|------|
| popularity_rank | 2025-03-15 ~ 2026-03-20 | 184,993 | Top500热门股/日，371天 |
| market_daily + adj_factor | 同上 | 1,260,944 | 5199只股票前复权行情 |
| index_daily | 同上 | 246 | 沪深300日线 |

**训练/测试切分**：2026-01-01 前为训练集（97,588条），之后为测试集（21,844条）

## 3. 特征工程（52个特征）

### 3.1 热度排名特征（26个）
- **排名本身**：rank, rank_pct（归一化）
- **排名变化**：rank_chg_1d/3d/5d/10d（短中期排名动量）
- **排名统计**：rank_ma5/10/20, rank_std5/10（均值与波动）
- **热度位置**：heat_position_10/20/30（近N日范围内归一化位置）
- **排名弹性**：rank_surge_10/20（偏离均值程度）
- **热度持续性**：days_in_top100/500（近20天进入TopN天数）
- **极值**：rank_min_20d, rank_max_20d
- **量价原始值**：change_rate, volume_ratio, turnover_rate, deal_amount

### 3.2 行情特征（14个）
- 收益率：return_1d/3d/5d/10d
- 技术指标：amplitude, ma_bias_5/10/20, vol_ratio_5/10, price_pos_20/60, volatility_10/20

### 3.3 大盘环境特征（9个）
- 沪深300：index_return_1d/5d/10d, index_volatility_10, index_ma_bias_20
- 市场热度分布：market_heat_mean/std/skew, top100_avg_chg

### 3.4 交叉特征（3个）
- heat_momentum_cross, cold_with_volume, heat_vol_interaction

## 4. 模型与结果

### 4.1 回归任务（预测5日收益率）

| 指标 | LightGBM | XGBoost | LSTM (GPU) |
|------|----------|---------|------------|
| IC(Pearson) | **0.0657** | 0.0512 | 0.0550 |
| IC(Spearman) | 0.0440 | 0.0216 | **0.0753** |
| ICIR | **1.18** | 1.11 | — |
| IC>0 比例 | **88.64%** | 86.36% | — |
| MSE | **0.0130** | 0.0132 | 0.0217 |
| 方向准确率 | 49.6% | 48.4% | **50.9%** |
| 多空收益 | 1.08% | 0.76% | **1.89%** |
| 训练时间 | **0.8s** | 2.6s | 39.4s |

**关键发现**：
- **ICIR=1.18 达到优秀水平**（>1.0 为行业标准优秀线），预测稳定性好
- **IC>0 比例 88.64%**，44个交易日中仅5天 IC 为负，时间稳定性极强
- **LSTM 的 Spearman IC 最高**（0.075），说明热度数据存在时序模式
- 多空收益方面 LSTM（1.89%）> LightGBM（1.08%）> XGBoost（0.76%）

### 4.2 分类任务（预测5日涨跌方向）

| 指标 | LightGBM | XGBoost |
|------|----------|---------|
| AUC | 0.512 | **0.522** |
| 准确率 | 0.544 | **0.551** |
| 精确率 | 0.425 | 0.434 |
| 召回率 | 0.319 | 0.306 |

**分类任务效果一般**：AUC 仅略高于0.5，说明热度特征对方向判断的贡献有限，但对收益幅度的区分力（回归IC）更好。

## 6. 核心发现与洞察

### 6.1 大盘主导效应

**最重要的发现是：个股未来收益的主要驱动力来自大盘环境，而非个股热度本身。**

SHAP分析显示，前3名特征全部是大盘指标（index_ma_bias_20, index_return_10d, index_volatility_10），占据近48%的预测贡献。这意味着：
- 在牛市环境中，高热度股票整体收益更好
- 在震荡市中，模型更多依赖行情技术面特征
- 热度排名特征是有效的，但需要在大盘环境条件下才能更好发挥

### 6.2 热度特征的有效性

虽然热度特征不是最强特征，但确实有独立预测价值：
- `rank_min_20d`（近20天最高排名）的 SHAP 值 0.00141，说明历史热度峰值包含有效信息
- `rank_ma20`（排名20日均值）作为中期热度水平的代理变量也有贡献
- 热度位置（heat_position）系列特征的贡献相对较弱

### 6.3 LSTM 的优势

LSTM 的 Spearman IC（0.075）显著高于树模型（0.044），说明：
- 热度数据中存在**时序依赖**模式，单日截面特征无法完全捕获
- GPU 加速的深度学习模型在处理序列模式时更有优势
- 可以进一步探索 Transformer 等更先进的时序架构

### 6.4 回归优于分类

回归任务（ICIR=1.18）远优于分类任务（AUC=0.52），说明：
- 热度特征更擅长区分**收益幅度**而非**涨跌方向**
- 适合做排序因子（选出收益最高/最低的股票），不适合做方向预测
- 这与多因子选股框架天然契合

## 7. 后续方向

| 方向 | 具体内容 | 预期价值 |
|------|----------|---------|
| **多时间周期** | 对 3/5/10 天分别训练，比较不同周期预测力 | 找到最佳预测周期 |
| **大盘条件分层** | 按牛/熊/震荡分别训练模型 | 消除大盘主导效应 |
| **时序模型深化** | Transformer, GRU, 加入 attention 机制 | 捕获更复杂的时序模式 |
| **融合情感数据** | 加入 guba_sentiment, news_analysis 特征 | 多维度情绪信号 |
| **实盘信号** | 将模型预测值集成到 live_signal.py | 实际应用 |
| **特征选择** | 去除冗余特征，降维后重新训练 | 减少过拟合风险 |
