# LA 选股分析脚本说明

## 脚本列表

### analyze_model_performance.py — 模型综合评估
**功能**: 分析 LA 选股各模型的预测能力，生成多维度评估报告

**分析维度**:
- 市场行情概览（大盘趋势）
- 各模型整体表现（胜率、均收益）
- 做多 vs 做空准确性与幅度
- 各策略方法论表现对比
- 预测时间有效性（T+1 / T+2 / T+5）
- 逐日选股表现
- 数据质量说明

**用法**:
```bash
# 在 my-stock 目录下运行（需 PYTHONPATH 指向 my-stock）
PYTHONPATH=F:/projects/my-stock python la/scripts/analyze_model_performance.py

# 输出到文件
python la/scripts/analyze_model_performance.py --output la/report/20260321-分析报告.md
```

### analyze_picks.py — 选股结果浏览
查看 la_pick 表的原始数据

### analyze_characteristics.py — 选股特征分析
分析选股的行业/市值/评分特征

### analyze_consistency.py — 模型一致性分析
分析不同模型选同一股票的一致性

### analyze_divergence.py — 分歧分析
分析模型间分歧较大的股票

### backfill_return.py — 收益率回填
**功能**: 从 market_daily 读取行情，回填 la_pick 的 T+N 收益率

**用法**:
```bash
# 回填所有未完成记录
python la/scripts/backfill_return.py

# 只回填指定日期
python la/scripts/backfill_return.py --eval-date 20260312
```
