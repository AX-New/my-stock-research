# 语雀文档目录约束
**创建时间**: 20260318

使用 `scripts/yuque_note.py` 操作，用法见 `python scripts/yuque_note.py --help`。

## 上传规则

1. 标题形式： `010-{日期}-简介`
2. **必须上传至以下目录**，禁止不设置目录，目录结构变化，按序号排列
  - `030-指标分析` -> 指标分析研究报告
  - `040-量化框架` -> 细化系统设计及总结报告
  - `050-LLM预测`
    - `020-对比分析` -> la功能 结果分析报告
  - `060-量化策略`
    - `010-策略研究` -> 研究报告、方法论、代码
    - `020-结果分析` -> 回测结果分析报告
  - `070-工程文档`
    - `010-系统设计` -> 系统架构设计文档
    - `020-任务清单` -> 计划任务机进度清单
    - `030-经验总结` -> 经验总结、踩坑记录
    - `040-其他` -> 其他不分类的文档
