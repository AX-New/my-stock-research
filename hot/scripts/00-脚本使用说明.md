# 热度排名研究 - 脚本说明

| 脚本 | 用途 |
|------|------|
| `verify_heat_hypotheses.py` | 热度排名与股价关系验证（飙升反转/持续高热/滞后指标/极端冷门/排名变化共5个假说） |
| `calc_excess_returns.py` | 计算热度分组相对三大指数（上证/深证/创业板）的超额收益 |
| `analyze_heat_velocity.py` | 热度涨速与股价、资金流向关系分析（涨速vs收益/涨速vs资金/多因子IC分析），支持 `--part 1\|2\|3\|all` |

## 回测脚本（task/162-热度策略回测/）

| 脚本 | 用途 |
|------|------|
| `01_heat_backtest.py` | v1：逐股独立回测（每只股票独立买卖，统计全市场平均表现） |
| `02_heat_rotation_backtest.py` | **v2：单仓位全市场轮转回测**（模拟真实操作：全仓1只股票，卖出后立即换入下一只最合适的，使用rank_surge选股） |
