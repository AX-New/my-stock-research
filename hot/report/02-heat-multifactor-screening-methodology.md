# 热度+多因子选股方法论

> 基于热度排名研究结论，结合技术面+基本面的三层筛选框架

---

## 一、核心思路

```
第一层: 热度过滤 → 从5000+只股票中筛出冷门区（排名>3500）
第二层: 技术确认 → 从冷门股中找趋势向好的（MACD/MA/RSI）
第三层: 基本面验证 → 从技术达标的中选财务健康的
```

热度因子是**负向过滤器**（排除热门）而非正向选股器（不是"越冷越好"），必须叠加其他维度才有实操价值。

---

## 二、第一层：热度过滤

### 依据

来自 `hot/report/01-heat-rank-hypothesis-verification.md` 的核心结论：

| 热度区间 | T+20胜率 | T+60胜率 | T+120胜率 | 判定 |
|---------|---------|---------|----------|------|
| 热门(1-200) | 40.2% | 46.0% | 52.1% | 回避 |
| 中性(501-2000) | 51.5% | 58.7% | 66.7% | 中性 |
| 冷门(>3500) | 61.5%~64.6% | 69.1%~75.3% | 78.0%~84.6% | 优选 |

### 执行规则

```sql
-- 数据源: my_trend.em_hot_rank_detail（最新交易日）
WHERE rank > 3500           -- 冷门区（极冷+冰封）
  AND ts_code NOT LIKE '688%'  -- 按需排除科创板（如果无权限）
```

### 附加规则：热门股卖出信号

```
持仓股连续2天以上进入 top 100 → 减仓50%
持仓股连续3天以上进入 top 100 → 清仓
排除常驻热门股（东方财富/茅台等年度top100天数>100的）
```

## 三、第二层：技术面确认

### 目的

冷门股中大量是"冷门且在跌"的股票（应该冷门），需要技术面筛出"冷门但趋势向好"的（被低估的）。

### 信号选择依据

来自 MACD/RSI L4 个股研究的核心结论：

| 信号类型 | 个股胜率 | 判定 | 是否采用 |
|---------|---------|------|---------|
| MACD金叉(DIF>DEA) | 49.8% | **噪音** | ❌ 禁用 |
| MACD死叉(DIF<DEA) | 53.5% | 噪音 | ❌ 禁用 |
| DIF极小值(谷底) | **83~97%** | 极强买入 | ✅ 核心信号 |
| DIF极大值(峰顶) | **83~97%** | 极强卖出 | ✅ 卖出用 |
| 底背离 | **99.7~100%** | 完美但极稀有 | ✅ 若出现必买 |
| RSI极端低值(<30) | **>70%** | 强买入 | ✅ 核心信号 |

**关键教训**：交叉类信号（金叉/死叉/零轴穿越）在个股级别全部退化为噪音（48~52%），只有极端值和背离有实操价值。

### 筛选策略：极值反转型

寻找冷门股中刚从极端超卖恢复的股票（DIF 谷底 + RSI 极端低值双重共振）。

```sql
-- 核心: DIF为负且拐头回升（接近极小值）
AND f_today.macd_dif_qfq < 0                         -- DIF处于负值区（弱势）
AND f_today.macd_dif_qfq > f_yesterday.macd_dif_qfq  -- DIF开始回升（拐头向上）

-- 核心: RSI近5天触及过极端超卖后恢复
AND rsi_5d_min < 30                                   -- 近5天RSI6曾低于30（极端超卖）
AND f_today.rsi_qfq_6 > 30                            -- 当前已回升（确认不是持续暴跌）
```

**逻辑**：
- DIF负值+拐头 = 下跌动量衰竭，接近 DIF 极小值（个股胜率83%+）
- RSI触及<30后恢复 = 极端超卖反转（个股胜率>70%）
- 两个信号共振，互相验证，提高可靠性
- **不要求站上均线**：极值信号的要义就是"在底部买"，强制要求站上MA20/MA60会错过最佳入场点

### 可选加强信号（不强制）

```sql
-- 若近期出现底背离（价格新低但DIF未新低），信号升级为S级
-- 底背离检测较复杂，需回溯60天数据对比前一轮DIF低点

-- 距MA60不超过-15%（跌太深可能有基本面问题）
AND close_qfq / ma_qfq_60 > 0.85
```

### 通用过滤

```sql
AND total_mv > 300000                  -- 市值>30亿（排除微盘垃圾股）
AND name NOT LIKE '%ST%'               -- 非ST
AND turnover_rate > 0.5                -- 有基本流动性
AND pe_ttm > 0 AND pe_ttm < 40~50     -- 盈利且估值不离谱
AND pb > 0 AND pb < 5~6               -- PB合理
```

## 四、第三层：基本面验证

技术面通过后，逐只查财务数据做深度筛选。这一步无法完全自动化，需要人工判断。

### 核心指标（按重要性排序）

| 优先级 | 指标 | 达标标准 | 数据源 | 一票否决线 |
|-------|------|---------|--------|-----------|
| ★★★ | 经营现金流 | 近3年多数为正，且与净利匹配 | finance_cashflow | 连续3年为负 → 淘汰 |
| ★★★ | 营收趋势 | 近3年增长或至少不萎缩 | finance_income | 连续2年下滑>15% → 警惕 |
| ★★★ | 净利润趋势 | 近3年增长 | finance_income | 最近年度亏损 → 淘汰 |
| ★★ | 毛利率 | >25%（有产品溢价） | finance_fina_indicator | <15% → 低附加值，谨慎 |
| ★★ | 资产负债率 | <50% | finance_fina_indicator | >65% → 淘汰 |
| ★★ | ROE | >8%（资本回报率） | finance_fina_indicator | <3% → 效率太低 |
| ★ | 股东户数 | 稳定或减少（筹码集中） | stk_holdernumber | 快速增加 → 散户涌入 |
| ★ | PE/PB | PE<30, PB<3 为佳 | stk_factor_pro | 仅作参考，不单独判断 |

### 评分维度

将基本面归纳为4个维度打分：

| 维度 | 含义 | 关键指标 |
|------|------|---------|
| **增长性** | 业务是否在扩张 | 营收增速、净利增速、2025Q3 同比 |
| **盈利质量** | 赚的钱是否是真钱 | 现金流/净利润比率、毛利率 |
| **财务安全** | 抗风险能力 | 负债率、流动比率、现金储备 |
| **估值吸引力** | 当前价格是否便宜 | PE、PB、vs 历史分位 |

### 组合风格标签

根据四维度得分，给每只股票贴风格标签：

| 风格 | 特征 | 适合 |
|------|------|------|
| **稳健成长** | 增长+质量均好，估值合理 | 核心仓位 |
| **成长优先** | 增速高但估值偏贵或波动大 | 进攻仓位 |
| **防守型** | 增长慢但安全边际极高（低PB低负债） | 底仓/防守 |
| **高弹性** | 利润反转或爆发，但持续性存疑 | 卫星仓位，严控比例 |

## 五、组合构建原则

### 持仓周期

热度因子在 **T+60 ~ T+150 效果最强**，建议持有 **3~6个月**。

### 分散规则

- 选 3~5 只，不超过 5 只
- 行业不重复（避免同涨同跌）
- 风格搭配：至少 1 只稳健 + 1 只成长 + 1 只防守

### 卖出规则

| 信号 | 动作 |
|------|------|
| 热度排名进入 top 200 | 减仓 50%（从冷变热，alpha 消失） |
| 连续 2 天进入 top 100 | 清仓 |
| 持有满 6 个月 | 重新评估，不盲目续持 |
| 基本面恶化（季报营收下滑>20%） | 清仓 |

### 入场时机

- 优先在靠近 MA20 支撑时买入（vs MA20 < 3%）
- 避免在 RSI > 70 时追入
- 大盘暴跌日可能是批量入场时机（冷门股跌幅通常更小）

## 六、实操示例（2026-03-12 筛选）

### 筛选漏斗

```
全量A股                      ~5500只
→ 热度排名 > 3500            ~1800只（冷门区）
→ 排除科创板                 ~1200只
```

## 七、方法论局限

| 局限 | 说明 |
|------|------|
| 热度数据仅1年 | 未经熊市验证，冷门股溢价可能在熊市消失或反转 |
| 市值混淆 | 冷门偏小盘，部分alpha可能来自小盘因子 |
| 选股频率低 | 冷门+技术达标+基本面好的交集很小（~4只/次） |
| 无回测 | 方法论尚未用历史数据做系统化回测验证 |
| 科创板限制 | 排除688后候选池大幅缩小 |
