# RSI指标研究 - 脚本说明

## 计算脚本

| 脚本 | 用途 |
|------|------|
| `compute_index_rsi.py` | 指数/申万行业RSI全市场计算（RSI 6/12/14/24四个周期） |
| `compute_stock_rsi.py` | 个股RSI全市场计算（4频率x3复权=12张表） |
| `compute_stock_rsi_signals.py` | 个股RSI信号事件回填，检测极端值/背离/失败摆动/中轴穿越 |

## 分析脚本

| 脚本 | 用途 |
|------|------|
| `analyze_index_rsi.py` | Layer 1-2: 指数RSI信号全维度分析（分布特征 + 信号统计 + 牛熊交叉） |
| `analyze_stock_rsi.py` | Layer 4: 全市场个股RSI信号统计分析，汇总写入 stock_rsi_signal_stats |
| `analyze_sw_rsi.py` | Layer 3: 申万行业RSI信号分析，按5大类别聚合对比 |
| `analyze_broadbase_rsi.py` | Layer 2.5: 四大宽基指数RSI市值分层验证（上证50/沪深300/中证500/中证1000） |
| `analyze_time_stability_rsi.py` | Layer 3.5: RSI信号时间稳定性验证，按5年分段检查方向稳定性和幅度衰减 |

## 基础模块

| 脚本 | 用途 |
|------|------|
| `config.py` | 数据库连接配置（读 my_stock / 写 stock_rsi） |
| `database.py` | 双引擎初始化（read_engine + write_engine） |
| `db_utils.py` | batch_upsert 批量写入工具 |
| `kline_loader.py` | K线数据加载，复用MACD模块的加载函数 |
| `models.py` | RSI表模型定义（28张表：数据表 + 信号表 + 统计表） |
| `rsi_calc.py` | RSI计算引擎（Wilder平滑法） |
| `signal_detector_rsi.py` | RSI信号检测引擎（极端值/背离/失败摆动/中轴穿越） |
| `bull_bear_phases.py` | 牛熊周期数据，从MACD研究模块复用 |
