# 波动率研究基础
**创建时间**: 20260319

## 研究目标

预测个股日内价格波动区间，用于精确设定买入/卖出限价单。

## 核心指标: ATR (Average True Range)

### 计算公式

```
TR = max(High - Low, |High - PrevClose|, |Low - PrevClose|)
ATR(N) = Wilder Smoothing(TR, N)
ATR_ratio = ATR / Close × 100%
```

### 推荐参数

- **ATR(14)**: 日线标准周期，覆盖率与灵敏度最佳平衡
- **ATR(5)**: 短期波动敏感，适合高频交易

### 关键结论（已验证）

1. ATR×1.0 覆盖率约 65%（次日振幅 ≤ ATR 的概率）
2. ATR×1.5 覆盖率约 89%
3. ATR 在牛熊市都保持有效（自适应指标）
4. 买入侧偏离中位数 ≈ 0.35×ATR，卖出侧 ≈ 0.37×ATR

## 数据来源

- K线数据: `my_stock` 库 `market_daily`（个股）/ `index_daily`（指数）
- 复权: 个股用前复权 (qfq)，指数无需复权
- 行业: `index_member_all` 表（申万分类）

## 写入库

`stock_volatility`（当前阶段未落库，分析结果输出为 CSV）
