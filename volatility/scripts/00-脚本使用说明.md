```bash
# 必须使用 my-stock conda 环境
# 必须设置 PYTHONPATH 包含 my-stock 项目和 macd/scripts
cd volatility/scripts
PYTHONPATH="F:/projects/my-stock;F:/projects/my-stock-research-wt-claude03/volatility/scripts;F:/projects/my-stock-research-wt-claude03/macd/scripts" \
  "D:\Program Files\anaconda3\envs\my-stock\python.exe" <脚本名>
```

## 脚本列表

### 基础设施

| 脚本 | 说明 |
|------|------|
| `config.py` | 双库连接配置（读 my_stock / 写 stock_volatility） |
| `database.py` | 数据库引擎（自动建库） |
| `kline_loader.py` | K线加载（复用 MACD 模块） |
| `volatility_calc.py` | 波动率指标计算（TR/ATR/ADR/HV/BBW） |

### 分析脚本

| 脚本 | 说明 | 输出 |
|------|------|------|
| `analyze_index_volatility.py` | 7大指数波动率分析（Layer 1-2），含覆盖率、多指标对比、牛熊分组、买卖偏离 | `data/index_*.csv` |
| `analyze_stock_volatility.py --mode batch` | 全市场个股抽样分析，按行业/市值分组统计 | `data/stock_batch_volatility.csv` |
| `analyze_stock_volatility.py --mode single --code XXX` | 单股详细波动率分析，各ATR周期覆盖率+买卖偏离 | 终端输出 |
| `analyze_stock_volatility.py --mode predict --code XXX` | **实用工具**: 输出具体买卖建议价和置信区间 | 终端输出 |

## 使用示例

```bash
# 1. 指数分析（约6秒）
python analyze_index_volatility.py

# 2. 批量个股分析（300只约80秒）
python analyze_stock_volatility.py --mode batch --sample 300

# 3. 单股分析
python analyze_stock_volatility.py --mode single --code 600519.SH

# 4. 买卖预测（日常使用）
python analyze_stock_volatility.py --mode predict --code 600519.SH
```
