# 热度轮转策略回测报告（双条件出场：热度回升+DIF见顶）
**创建时间**: 20260320 07:03

## 一、回测逻辑说明

### 1.1 策略思路
利用东财人气排名的均值回归特性选股，用热度回升+DIF见顶双条件判断出场时机。
当一只股票的热度跌到近期最低点时买入，当热度回升且DIF出现见顶信号时卖出。
**双条件必须在同一天同时满足才触发卖出**，避免过早离场。

### 1.2 交易流程
```
每个交易日收盘后：
  1. 如果持仓中 → 检查是否同时满足双条件卖出
     - 条件A: heat_position <= 0.2（热度已回升到高位）
     - 条件B: DIF连降2根（M4信号，DIF见顶）
     - 两者必须在同一天同时满足
     - 无超时强制平仓
  2. 如果无持仓（初始/刚卖出） → 全市场选股
     - 筛选heat_position >= 0.8的股票（热度处于近期低谷）
     - 过滤日成交额 < 0.5亿的小票
     - 在候选中选rank_surge最高的（相对自身常态冷了最多的）
  3. 次日收盘价执行买入/卖出
     - 卖出和买入可以同日执行（卖旧买新）
```

### 1.3 双条件出场的设计思路

单独的DIF出场可能在热度还没回升时就过早卖出（趋势还在酝酿中）。
单独的热度出场可能在股价还在上涨时就卖出（热度回升但动量未衰减）。

双条件出场的逻辑：
- **热度回升**（heat_position <= 0.20）：市场注意力已经回来，均值回归的"燃料"消耗殆尽
- **DIF见顶**（连降2天）：价格上涨动量已经衰减，趋势可能反转
- 两者同时满足 = 该赚的钱已经赚到了，是时候离场

### 1.4 核心指标

**heat_position（入场信号）：**
- 取过去20个交易日的热度排名（rank）
- heat_position = (当前rank - N日最低rank) / (N日最高rank - N日最低rank)
- 值域 [0, 1]，越接近1表示热度越冷（rank越大=越不受关注）

**MACD DIF（出场信号之一）：**
- 使用前复权收盘价计算MACD (EMA12/26, signal=9)
- DIF = EMA_short - EMA_long
- DIF见顶判断（M4）：DIF连续2天下降 → 趋势转弱

### 1.5 资金管理
- 初始资金: 100万
- 每次全仓买入1只股票
- 卖出后全部资金立即投入下一只
- 未计入交易手续费
