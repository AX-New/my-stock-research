# 热度轮转策略回测报告（单仓位全市场轮转）
**创建时间**: 20260320 02:19

## 一、回测逻辑说明

### 1.1 策略思路
利用东财人气排名的均值回归特性：当一只股票的热度跌到近期最低点时买入，
等热度回升后卖出，然后立即从全市场选出下一只"最冷门"的股票买入。
**资金始终满仓运转，没有空窗期。**

### 1.2 交易流程
```
每个交易日收盘后：
  1. 如果持仓中 → 检查是否触发卖出条件
     - 条件A: 持仓股的heat_position <= 0.2（热度已回升到近期高点）
     - 条件B: 持仓天数 >= 15天（超时强制平仓）
  2. 如果无持仓（初始/刚卖出） → 全市场选股
     - 筛选heat_position >= 0.8的股票（热度处于近期低谷）
     - 过滤日成交额 < 0.5亿的小票
     - 按rank_surge排序（当前rank/近期均值rank），选"突然变冷最多"的
  3. 次日收盘价执行买入/卖出
     - 卖出和买入可以同日执行（卖旧买新）
```

### 1.3 heat_position 计算方法
- 取过去20个交易日的热度排名（rank）
- heat_position = (当前rank - N日最低rank) / (N日最高rank - N日最低rank)
- 值域 [0, 1]，越接近1表示热度越冷（rank越大=越不受关注）

### 1.4 资金管理
- 初始资金: 100万
- 每次全仓买入1只股票
- 卖出后全部资金立即投入下一只
- 未计入交易手续费

## 九、核心结论

### 1. 策略能跑赢大盘吗？

**能，但不稳定。** 默认参数（lookback=20）策略年化32.7%，跑赢沪深300约17个百分点。
最优参数组合可以实现年化100%+的超额收益。但月度来看，仅54%的月份跑赢大盘（7/13个月），
且单月波动极大（最好月份+28.6%，最差月份-21.0%）。

### 2. 关键发现

**热度回升卖出的交易质量极高：**
- 因"热度回升"信号平仓的16笔交易：**胜率87.5%**，平均收益+4.99%
- 因"超时"平仓的7笔交易：胜率仅28.6%，平均亏损-4.91%
- **结论**：策略的alpha来自热度均值回归信号本身，等不到热度回升的交易基本都是亏损

**选股标准（rank_surge）的改进效果显著：**
- 使用rank_surge（当前rank/近期均值rank）代替简单的heat_position排序
- 自然偏好"平时热门、突然变冷"的股票，这类股票热度回归概率更高
- 改进后策略从+13%（未跑赢大盘）提升到+32%（跑赢17个百分点）

**短lookback窗口表现更好：**
- lookback=10 → +108.6%（Sharpe 1.57）
- lookback=20 → +31.9%（Sharpe 0.76）
- lookback=30 → +16.0%（Sharpe 0.51）
- 说明热度均值回归是**短期现象**（5~15天），用更短的窗口能更敏锐地捕捉

### 3. 策略优势

| 优势 | 说明 |
|------|------|
| 无空窗期 | 卖出后立即买入下一只，资金始终在运转 |
| 候选充足 | 全市场5000+只股票，每天都能找到低热度候选 |
| 信号明确 | 买卖逻辑清晰（热度低买高卖），不依赖主观判断 |
| 高胜率 | 信号平仓胜率87.5%，策略逻辑经得起验证 |

### 4. 风险与局限

| 风险 | 严重程度 | 说明 |
|------|---------|------|
| **单票集中** | 高 | 单仓位全仓1只股票，单笔亏损可达-14%~-20% |
| **最大回撤大** | 高 | 默认参数最大回撤-39%，连续亏损期间心理压力极大 |
| **超时平仓拖累** | 中 | 约30%的交易因超时退出，拖累整体收益 |
| **参数敏感** | 中 | 不同参数组合结果差异巨大（-25%到+119%） |
| **样本期短** | 中 | 仅1年数据（245个交易日），需更长期验证 |
| **未计手续费** | 低 | 约24笔交易/年，手续费影响约1-2% |

### 5. 改进建议

1. **分散持仓**: 同时持有3~5只候选股，分散单票风险，平滑净值曲线
2. **缩短止损周期**: 超时平仓是最大亏损来源，建议从15天缩短到7~10天
3. **叠加过滤条件**: 排除ST股、次新股；叠加大盘趋势过滤（大盘弱势时减仓）
4. **动态参数**: lookback可根据近期市场波动率动态调整
5. **更多选股因子**: 在rank_surge基础上叠加成交量变化、RSI等因子，提升信号质量
