**创建时间**: 20260320 07:17

## 一、回测逻辑说明

### 1.1 策略思路
选择低波动率整理、连续下跌、且热度冷门的股票买入（平台整理后的冷门回调股）。
用热度回升+DIF见顶双条件判断出场时机。
**核心假设**：冷门低波动股在连续下跌后容易触发反弹，热度回升+DIF转弱时正好是获利了结的时机。

### 1.2 入场条件（三条件同时满足）

```
条件1: 20日ATR/close < 0.03 — 低波动率整理
        ATR = rolling 20-day mean of (qfq_high - qfq_low)
        ATR_ratio = ATR / qfq_close
条件2: 连续3天下跌 — close[t]<close[t-1]<close[t-2]<close[t-3]
条件3: 热度rank > 2000 — 不在最热门的2000只里
过滤:  日成交额 >= 0.5亿
排序:  rank_surge 最高优先（相对自身常态冷了最多的）
```

### 1.3 出场条件（双条件同时满足）

```
条件A: heat_position <= 0.2（热度已回升到高位）
条件B: DIF连降2天（M4信号，DIF见顶）
两者必须在同一天同时满足才触发卖出
无超时强制平仓（MAX_HOLD_DAYS = 9999）
期末强制平仓
```

### 1.4 交易流程
```
每个交易日收盘后：
  1. 如果持仓中 → 检查是否同时满足双条件卖出
  2. 如果无持仓 → 全市场三条件选股
  3. 次日收盘价执行买入/卖出
```

### 1.5 资金管理
- 初始资金: 100万
- 每次全仓买入1只股票
- 卖出后全部资金立即投入下一只
- 未计入交易手续费
