# 热度轮转策略回测报告（金叉后死叉出场）
**创建时间**: 20260320 07:03

## 一、回测逻辑说明

### 1.1 策略思路
利用东财人气排名的均值回归特性选股，用MACD金叉/死叉信号判断出场时机。
当一只股票的热度跌到近期最低点时买入，等待DIF上穿DEA（金叉确认趋势启动），
再等待DIF下穿DEA（死叉确认趋势结束）时卖出，然后立即轮转到下一只。
**资金始终满仓运转，没有空窗期。**

### 1.2 交易流程
```
每个交易日收盘后：
  1. 如果持仓中 → 两阶段出场判断
     Phase 1（等金叉）：
       - DIF_yesterday <= DEA_yesterday AND DIF_today > DEA_today → 金叉确认
       - 标记 golden_cross_seen = True
     Phase 2（等死叉，仅在金叉后）：
       - DIF_yesterday > DEA_yesterday AND DIF_today <= DEA_today → 死叉确认
       - 触发卖出，次日执行
     无超时强制平仓
  2. 如果无持仓（初始/刚卖出） → 全市场选股
     - 筛选heat_position >= 0.8的股票（热度处于近期低谷）
     - 过滤日成交额 < 0.5亿的小票
     - 在候选中选rank_surge最高的（相对自身常态冷了最多的）
  3. 次日收盘价执行买入/卖出
     - 卖出和买入可以同日执行（卖旧买新）
```

### 1.3 核心指标

**heat_position（入场信号）：**
- 取过去20个交易日的热度排名（rank）
- heat_position = (当前rank - N日最低rank) / (N日最高rank - N日最低rank)
- 值域 [0, 1]，越接近1表示热度越冷（rank越大=越不受关注）

**MACD 金叉/死叉（出场信号）：**
- 使用前复权收盘价计算MACD (EMA12/26, signal=9)
- DIF = EMA_short - EMA_long，DEA = EMA(DIF, 9)
- 金叉：DIF从下方穿越DEA到上方（趋势启动）
- 死叉：DIF从上方穿越DEA到下方（趋势结束）
- 必须先看到金叉，再等死叉出场 → 确保完整趋势周期

### 1.4 资金管理
- 初始资金: 100万
- 每次全仓买入1只股票
- 卖出后全部资金立即投入下一只
- 未计入交易手续费
