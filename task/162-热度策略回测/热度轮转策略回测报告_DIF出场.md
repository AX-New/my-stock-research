# 热度轮转策略回测报告（DIF极值出场）
**创建时间**: 20260320 06:47

## 一、回测逻辑说明

### 1.1 策略思路
利用东财人气排名的均值回归特性选股，用MACD DIF指标判断出场时机。
当一只股票的热度跌到近期最低点时买入，当DIF出现见顶信号（连续下降）时卖出，
然后立即从全市场选出下一只"最冷门"的股票买入。
**资金始终满仓运转，没有空窗期。**

### 1.2 交易流程
```
每个交易日收盘后：
  1. 如果持仓中 → 检查是否触发DIF见顶卖出条件
     - DIF连降2根（M4信号）：
       DIF_today < DIF_yesterday AND DIF_yesterday < DIF_day_before
     - 持仓至少1天后才检查（避免刚买入就因历史DIF下降卖出）
     - 无超时强制平仓
  2. 如果无持仓（初始/刚卖出） → 全市场选股
     - 筛选heat_position >= 0.8的股票（热度处于近期低谷）
     - 过滤日成交额 < 0.5亿的小票
     - 在候选中选rank_surge最高的（相对自身常态冷了最多的）
  3. 次日收盘价执行买入/卖出
     - 卖出和买入可以同日执行（卖旧买新）
```

### 1.3 核心指标

**heat_position（入场信号）：**
- 取过去20个交易日的热度排名（rank）
- heat_position = (当前rank - N日最低rank) / (N日最高rank - N日最低rank)
- 值域 [0, 1]，越接近1表示热度越冷（rank越大=越不受关注）

**MACD DIF（出场信号）：**
- 使用前复权收盘价计算MACD (EMA12/26, signal=9)
- DIF = EMA_short - EMA_long
- DIF见顶判断（M4）：DIF连续2天下降 → 趋势转弱 → 卖出

### 1.4 资金管理
- 初始资金: 100万
- 每次全仓买入1只股票
- 卖出后全部资金立即投入下一只
- 未计入交易手续费
