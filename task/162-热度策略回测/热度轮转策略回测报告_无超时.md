# 热度轮转策略回测报告（单仓位全市场轮转）
**创建时间**: 20260320 06:26

## 一、回测逻辑说明

### 1.1 策略思路
利用东财人气排名的均值回归特性：当一只股票的热度跌到近期最低点时买入，
等热度回升后卖出，然后立即从全市场选出下一只"最冷门"的股票买入。
**资金始终满仓运转，没有空窗期。**

### 1.2 交易流程
```
每个交易日收盘后：
  1. 如果持仓中 → 检查是否触发卖出条件
     - 条件A: 持仓股的heat_position <= 0.2（热度已回升到近期高点）
     - 条件B: 持仓天数 >= 9999天（超时强制平仓）
  2. 如果无持仓（初始/刚卖出） → 全市场选股
     - 筛选heat_position >= 0.8的股票（热度处于近期低谷）
     - 过滤日成交额 < 0.5亿的小票
     - 在候选中选heat_position最高的（最冷门的）
  3. 次日收盘价执行买入/卖出
     - 卖出和买入可以同日执行（卖旧买新）
```

### 1.3 heat_position 计算方法
- 取过去20个交易日的热度排名（rank）
- heat_position = (当前rank - N日最低rank) / (N日最高rank - N日最低rank)
- 值域 [0, 1]，越接近1表示热度越冷（rank越大=越不受关注）

### 1.4 资金管理
- 初始资金: 100万
- 每次全仓买入1只股票
- 卖出后全部资金立即投入下一只
- 未计入交易手续费
