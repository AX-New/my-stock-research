# 热度低买高卖策略回测报告
**创建时间**: 20260320 01:02

## 策略说明

**核心逻辑**: 当个股东财人气排名跌至近期低谷（热度低点，rank处于近期高位）时买入，排名回升至近期高峰（热度高点，rank处于近期低位）时卖出。

**数据基础**: 东财人气排名（`popularity_rank`），rank=1为全市场最热门，rank越大越冷门。每日覆盖约5,490只个股。

**信号定义**:
- 买入: 个股rank在过去20天滚动窗口内 >= 80%分位（热度处于近期低谷）
- 卖出: 个股rank在过去20天滚动窗口内 <= 20%分位（热度处于近期高峰）
- 最大持仓: 30个交易日，超时强制平仓
- 执行价格: 信号次日收盘价
- 回测区间: 2025-03-15 ~ 2026-03-19

## 核心结论

### 1. 策略有效性确认

热度均值回归策略在全市场回测中表现出**稳定的正期望值**。所有10组参数组合均取得正的平均收益（0.67%~2.29%）和超过50%的胜率（56.6%~64.3%），表明**股票热度的低买高卖逻辑在A股市场成立**。

### 2. 最优参数区间

| 维度 | 推荐值 | 依据 |
|------|--------|------|
| 回看窗口 | 20~30天 | 越长越准，但信号越少（胜率从59%→64%） |
| 买入阈值 | 0.85~0.90 | 更极端的冷门信号质量更高 |
| 卖出阈值 | 0.10~0.20 | 更极端的热门信号平仓更精准 |
| 最优持仓 | 3~4天 | 胜率72.2%，平均收益2.33%，超过10天收益显著下降 |

**最优参数组合**: lookback=30, buy=0.90, sell=0.10 → 胜率64.3%，平均每笔2.29%，但交易频率较低（47,198笔/年）。

### 3. 关键发现

- **短线策略属性**: 最佳持仓期3-4天，超过15天收益转负。热度均值回归是**短周期现象**，不适合中长期持仓。
- **信号平仓远优于超时平仓**: 信号平仓胜率61.0%/收益1.26%，超时平仓胜率仅8%/亏损9.38%。等不到卖出信号说明热度不回归，应设较短的止损期。
- **月度表现差异大**: 7月最强（胜率72%，均赚2.46%），11月、3月较弱。可能与市场活跃度、情绪周期有关。
- **盈亏比适中**: 平均盈利4.54% vs 平均亏损4.24%，盈亏比1.07，主要靠胜率（59%）取胜。

### 4. 风险提示

- 本回测假设每笔等金额投入，未考虑资金容量、涨跌停限制、停牌影响
- 2025年3月和2026年3月策略表现较差（胜率仅34-35%），说明**在市场大幅调整期策略失效**
- 未排除ST股和次新股，实际应用时应增加过滤条件
- 热度排名数据仅有1年历史，样本期偏短，需更长周期验证

### 5. 改进方向

1. **结合市场环境**: 在大盘下跌期减仓或暂停策略
2. **增加过滤条件**: 排除ST、次新股（<60天）、日成交额过低的股票
3. **动态止损**: 超过5天未触发卖出信号时主动止损
4. **与其他指标叠加**: 如叠加RSI超卖、资金净流入等条件提高信号质量
