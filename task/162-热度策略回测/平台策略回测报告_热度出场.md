# 平台突破策略回测报告（新入场+热度回升出场）
**创建时间**: 20260320 07:12

## 一、策略说明

### 1.1 策略思路
选择处于 **低波动平台整理** 中、**连续短期下跌**、且 **热度偏冷** 的股票买入，
等待热度回升后卖出。核心假设：低波动+冷门+短期回调 = 蓄势待发的均值回归机会。

与之前的热度轮转策略（v2无超时版）的主要区别：
- **旧入场**: heat_position >= 0.80（纯热度信号）
- **新入场**: ATR/close < 0.03 + 连续3天下跌 + rank > 2000（技术面+热度面复合信号）
- **出场条件相同**: heat_position <= 0.20（热度回升），无超时

### 1.2 入场条件（三条件同时满足）

| # | 条件 | 参数 | 含义 |
|---|------|------|------|
| 1 | 20日ATR/close < 0.03 | ATR = 20日rolling avg(high-low) | 低波动平台整理期 |
| 2 | 连续3天收盘价下跌 | close[t] < close[t-1] < ... | 短期回调 |
| 3 | 热度rank > 2000 | 东财人气排名 | 偏冷门股 |

### 1.3 交易流程
```
每个交易日收盘后：
  [持仓时] 检查是否触发卖出：
     - heat_position <= 0.2 → 热度已回升，卖出
     - 持仓天数 >= 9999 → 超时平仓（实际无限制）
  [空仓时] 全市场扫描：
     - 三条件筛选 → 流动性过滤(>= 0.5亿)
     - rank_surge最大的股票买入（偏离自身常态最远）
  执行：次日收盘价买入/卖出
```

### 1.4 资金管理
- 初始资金: 100万
- 每次全仓买入1只股票，卖出后全额投入下一只
- 未计入交易手续费
