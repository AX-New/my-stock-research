### 共振加分（指数级已验证，个股级待验证）

- DIF极小值 + 换手率暴增 → 收益翻倍（T+4w +8%→+16%）
- DIF极大值 + 换手率极端高 → 跌幅加深52%（T+4w -8%→-12%）
- MACD独行无共振的DIF极小值 → T+13w +22%（反直觉：最极端底部）
- 四指标全共振太稀缺（买入9次/卖出3次），不能作为常规依据

## 任务分解

### T1: 个股共振验证脚本

验证指数级共振结论在个股级是否成立。

- [ ] 从四张信号表联合查询，按 ts_code + trade_date ±N天窗口匹配共振
- [ ] 统计各共振组合的样本数、胜率、收益均值
- [ ] 与指数级结论对比，识别衰减幅度
- [ ] 输出到 `resonance/report/`

### T2: 选股扫描脚本

日常选股查询工具。

- [ ] 查询当日/近N天触发的信号，按优先级排序
- [ ] 支持按信号类型、行业、市值筛选
- [ ] 共振检测：每个 DIF 极值事件 ±2周内检查其他指标信号
- [ ] 输出共振评分 + 控制台报表

### T3: 推送集成（可选，视 T1/T2 结果决定）

如果个股共振验证有效：
