# 加密货币量化交易 - 脚本使用说明

## 架构概览

```
crypto/scripts/
├── config.py           ← 配置（交易所API、数据库、策略参数、风控）
├── database.py         ← 数据库连接（stock_crypto 库）
├── models.py           ← 数据模型（K线、信号、交易、持仓）
├── exchange_client.py  ← 交易所客户端（Binance/OKX，基于ccxt）
├── data_fetcher.py     ← 数据获取（历史K线抓取、增量更新、数据库读取）
├── indicators.py       ← 技术指标（MA/EMA/RSI/MACD/布林带/ATR）
├── strategy.py         ← 策略引擎（双均线/RSI/MACD/布林带/组合策略）
├── backtest.py         ← 回测引擎（止损止盈、统计报告）
├── trader.py           ← 交易执行（模拟/实盘、持仓管理）
└── run_bot.py          ← 自动交易机器人（主运行脚本）
```

## 前置条件

1. 安装依赖：`pip install ccxt`
2. 网络代理：需要代理访问 Binance/OKX（默认 localhost:7890）
3. API Key（可选）：仅实盘交易需要，配置在环境变量中

## 使用流程

### 1. 获取历史数据

```bash
# 获取 BTC/USDT 1小时K线，最近365天
python data_fetcher.py --symbol BTC/USDT --timeframe 1h --days 365

# 获取 ETH/USDT 4小时K线
python data_fetcher.py --symbol ETH/USDT --timeframe 4h --days 180

# 增量更新（只获取最新数据）
python data_fetcher.py --symbol BTC/USDT --timeframe 1h --update

# 使用 OKX 交易所
python data_fetcher.py --symbol BTC/USDT --timeframe 1h --exchange okx
```

### 2. 策略回测

```bash
# 双均线策略回测
python backtest.py --symbol BTC/USDT --timeframe 1h --strategy dual_ma

# RSI 策略回测，10万USDT初始资金
python backtest.py --symbol BTC/USDT --timeframe 4h --strategy rsi --capital 100000

# MACD 策略，自定义止损止盈
python backtest.py --strategy macd --stop-loss 0.03 --take-profit 0.08

# 组合策略（多策略投票）
python backtest.py --strategy composite
```

### 3. 自动交易

```bash
# 模拟交易（默认），每5分钟检查一次
python run_bot.py --symbol BTC/USDT --strategy dual_ma --interval 300

# 多交易对
python run_bot.py --symbol BTC/USDT,ETH/USDT --strategy composite

# 沙盒模式（交易所测试网）
python run_bot.py --symbol BTC/USDT --strategy macd --sandbox

# 实盘交易（需要 API Key，会要求确认）
python run_bot.py --symbol BTC/USDT --strategy composite --live
```

## 策略说明

| 策略 | 名称 | 逻辑 |
|------|------|------|
| dual_ma | 双均线 | MA7上穿MA25买入，下穿卖出 |
| rsi | RSI | RSI<30超卖买入，RSI>70超买卖出 |
| macd | MACD | DIF上穿DEA金叉买入，死叉卖出 |
| bollinger | 布林带 | 触及下轨+RSI低买入，上轨+RSI高卖出 |
| composite | 组合投票 | 多策略投票，2票以上触发交易 |

## 环境变量配置

```bash
# 交易所 API（实盘必须）
BINANCE_API_KEY=xxx
BINANCE_API_SECRET=xxx

# 或 OKX
OKX_API_KEY=xxx
OKX_API_SECRET=xxx
OKX_PASSPHRASE=xxx

# 数据库
MYSQL_HOST=127.0.0.1
MYSQL_PORT=3307

# 代理
HTTP_PROXY=http://localhost:7890

# 默认交易所
CRYPTO_EXCHANGE=binance
```

## 数据库

写入库：`stock_crypto`（自动创建）

| 表 | 用途 |
|----|------|
| crypto_kline | K线数据（OHLCV） |
| crypto_signal | 策略信号记录 |
| crypto_trade | 交易执行记录 |
| crypto_position | 持仓管理 |
