# 加密货币量化系统 - CLI 脚本

所有脚本从 `crypto/` 目录执行。

## 数据同步

```bash
# 同步单个交易对K线
python scripts/sync_data.py --symbol BTC/USDT --timeframe 1h

# 全量同步（所有配置的交易对 × 所有周期）
python scripts/sync_data.py --all

# 仅同步交易对元数据
python scripts/sync_data.py --symbols-only

# 指定交易所和天数
python scripts/sync_data.py --symbol ETH/USDT --timeframe 4h --exchange okx --days 365
```

## 策略回测

```bash
# 默认：BTC/USDT 1h 双均线策略
python scripts/run_backtest.py

# 指定策略和参数
python scripts/run_backtest.py --symbol ETH/USDT --strategy macd --capital 50000

# 可用策略：dual_ma, rsi, macd, bollinger, composite
python scripts/run_backtest.py --strategy composite --stop-loss 0.03 --take-profit 0.08
```

## 自动交易

```bash
# 模拟交易（默认）
python scripts/run_bot.py --symbol BTC/USDT --strategy dual_ma

# 多交易对
python scripts/run_bot.py --symbol BTC/USDT,ETH/USDT --strategy macd --interval 60

# 实盘交易（需配置 API Key，会要求确认）
python scripts/run_bot.py --symbol BTC/USDT --strategy composite --live

# 沙盒模式（使用交易所测试网）
python scripts/run_bot.py --symbol BTC/USDT --live --sandbox
```

## API 服务

```bash
# 启动 API 服务（端口 8001）
python run.py

# 开发模式（自动重载）
python run.py --reload
```

API 文档: `http://localhost:8001/docs`

## 策略说明

| 策略 | 名称 | 逻辑 |
|------|------|------|
| dual_ma | 双均线 | MA7上穿MA25买入，下穿卖出 |
| rsi | RSI | RSI<30超卖买入，RSI>70超买卖出 |
| macd | MACD | DIF上穿DEA金叉买入，死叉卖出 |
| bollinger | 布林带 | 触及下轨+RSI低买入，上轨+RSI高卖出 |
| composite | 组合投票 | 多策略投票，2票以上触发交易 |

## 数据库

独立库：`stock_crypto`（自动创建）

| 表 | 用途 |
|----|------|
| crypto_symbol | 交易对元数据 |
| crypto_kline | K线数据（全量存储，无需复权） |
| crypto_signal | 策略信号记录 |
| crypto_trade | 交易执行记录 |
| crypto_position | 持仓管理 |
