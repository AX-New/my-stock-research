# 技术指标研究（以MACD为主）

- **状态**: IN_PROGRESS（L1~L5 全部完成，剩余应用层设计）
- **创建日期**: 2026-03-13

## 策略定位（已明确）

**价值投资为主体，技术指标做择时**，两者有机结合。

### 三层操作模式

**1. 长线持股（价值投资层）**
- 基于行业研究、长期趋势判断选股建仓
- 决策依据：基本面、行业前景，不是简单的技术指标
- 持股周期以年为单位

**2. 持股中波段套利（增强层）**
- 在长线持仓过程中，利用技术分析进行波段操作
- 目的：增厚利润 / 对冲错误决策产生的成本
- 参考周期：日线、月线、年线
- MACD角色：提供加仓、减仓的时机信号

**3. 纯技术套利（独立层）**
- 基于指标数据，纯技术面操作，打完就走
- 不依赖价值判断，纯粹的技术策略
- 参考周期：日线、月线、年线
- MACD角色：核心决策依据

### 分析原则

- **从顶底出发，反向研究指标特征**：不是看指标发出什么信号，而是先确认历史上的顶和底，再观察那些时刻各指标呈现什么特征
- 以MACD为主要研究对象，但不局限于MACD
- 允许结合其他指标/数据进行共振分析，多信号验证提高可靠性
- 单一指标产生的信号需要其他维度数据佐证

### 研究方法

```
传统思路（我们不用）：MACD金叉 → 预测上涨 → 验证准确率
我们的思路：        确认历史顶底 → 观察顶底时的指标特征 → 提炼共性规律
```

### 核心结论（全层级验证）

**1. 金叉死叉 ≈ 噪音（全层级一致）**
- 指数级日线胜率 42~54%，个股级 42~54%，接近随机
- 国际文献佐证：Meissner DOW30 研究 32%，Liberated Stock Trader 60万笔交易 26%
- 不能单独作为交易依据

**2. DIF 极值 — 最有实战价值的信号（全层级第一优先级）**
- 指数级：日线下跌率 85~94%，周线 88~100%，月线 100%
- 行业级：月线反转率 75~83%（低于指数但仍可用）
- 个股级：日线 89.9%，周线 **97.5%**，月线 **96.5%**（5473股验证）
- 个股平均收益：日线极大值后 T+20 **-11.05%**，极小值后 **+13.68%**；月线极大值后 T+3m **-21.41%**，极小值后 **+26.23%**
- **DIF 极值领先价格极值**：日线领先 3~21 天，周线 1~5 周（部分驳斥"MACD 滞后"说法，滞后的是金叉死叉，不是 DIF 拐点）
- 周线是最佳操作频率（反转率最高且频率适中，年均 3~4 次）

**3. 牛熊背景决定信号有效性**
- 牛市金叉(零轴下) T+60: +10~15%, 胜率 71~89%
- 熊市金叉(零轴下) T+60: -0~-4%, 胜率 34~44%
- **牛市死叉(零轴下) T+60: 94.7%后续上涨率, +18.1%** → 牛市死叉是加仓机会
- 同一信号在不同市场环境下完全相反

**4. 背离信号稀有但精准**
- 指数级：顶/底背离短期(T+5~T+20)胜率 100%（三指数一致），日线33年仅13~20次
- 个股级：严格定义下胜率 **99.9~100%**（年均每股仅4.3次）
- 注意：宽松定义（如知乎研究 7.1万次顶背离 54%）会严重稀释信号质量，定义严格程度决定一切

**5. 月线零轴均值回归 — A股普遍规律**
- 7个指数高度一致：极强多头(>Q75) → 未来1年 **-9.5%~-15.1%**，上涨率仅 20~30%
- 极强空头 → 未来1年 **+13.6%~+29.7%**，上涨率 67~100%
- **例外：大消费板块违反均值回归** — DIF>0 时1年 +13.5%（趋势延续），DIF<0 反而只有 +6.5%

**6. 市值效应 — 小盘弹性大**
- 月线 DIF 谷值后1年：上证50 +38.4% → 沪深300 +43.4% → 中证500 +49.2% → 中证1000 **+68.7%**
- 月线 DIF 峰值后1年：上证50 -20.2% → 中证1000 **-27.9%**
- 市值越小，上下振幅越大

**7. 行业分化显著**
- 科技成长弹性最大：月线 DIF 谷值后1年 **+45%**，是金融地产(+15%)的3倍
- 大消费违反均值回归：食品饮料/家电 DIF 峰值后仍涨 +5~7%（趋势延续型）
- 金融地产零轴位置无择时价值
- 周期股 MACD 信号最稳定，消费股信号最弱
- MACD 可靠行业 Top: 电力设备(100/100)、传媒(100/80)、机械(100/80)
- MACD 弱势行业: 零售(75/60)、国防(75/60)、医药(88/40)

**8. 共振分析 — 提升幅度而非胜率**
- MACD DIF 谷值单独使用时 T+4w 已达 **100%** 买入胜率，共振无法再提升
- 共振的价值在于筛选**更大幅度**的行情：
  - DIF 谷值 + 换手率暴增：T+4w **+16.27%**（基线 +8.26% 的近2倍）
  - DIF 峰值 + 超高换手：T+4w **-12.41%**（基线 -8.18%）
- 反直觉发现：孤立 DIF 谷值（无其他指标确认）T+13w **+22.05%** 最高 — 闪崩太快来不及共振，但恰恰标记最极端底部
- 四指标全共振极其稀有（历史仅9次买+3次卖），样本量不足以统计验证
- 2018-08-24 四指标共振买入信号失败（T+13w -5.17%）— 长期熊市降级所有信号

**9. 当前市场诊断（2026.03.14）**
- 月线 DIF = 241.0，处于 **93 百分位** — 深度过热
- 多频率背离：月线多头 vs 周线/日线已转空 — 经典顶部构型
- 历史顶部模板 **5/5 特征吻合**
- 牛市进度估算 **85%+**，类似2017末或2020末，剩余上行空间约 +5%
- 风格：小盘(中证500/1000)强于大盘(沪深300最弱)，周期+科技领涨，消费统一走弱
