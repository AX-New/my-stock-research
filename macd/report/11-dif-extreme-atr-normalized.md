# DIF极值信号ATR标准化分析
**创建时间**: 20260319 10:30

## 一、研究问题与方法

### 1.1 ATR标准化的动机

原报告（`10-dif-extreme-effectiveness.md`）使用原始 |DIF| 对极值信号进行幅度分层，存在**股价/波动率偏差**：
- 高价股（如贵州茅台 DIF 可达数百）天然拥有更大的 |DIF|，而低价股 |DIF| 普遍较小
- 同一只股票在高波动期和低波动期的 |DIF| 不可比
- 按原始 |DIF| 分位数分档，实际上是把高价股/高波股集中到 >Q75 档、低价股/低波股集中到 <Q25 档，幅度分层变成了"价格分层"

### 1.2 ATR标准化方法

**标准化指标 = |DIF| / ATR(20)**

- ATR(20)：20日平均真实波幅，反映该股近期的绝对波动水平
- DIF/ATR 的含义：DIF极值相当于多少倍的日内波动幅度
- 典型分布（个股日线）：Q25≈0.41, Q50≈0.84, Q75≈1.27，即大部分DIF极值幅度在0.4~1.3倍ATR之间

### 1.3 与原方法对比

| 维度 | 原方法（|DIF|分层） | ATR标准化（DIF/ATR分层） |
|------|-------------------|----------------------|
| 分层依据 | 绝对DIF值的四分位 | DIF/ATR比值的四分位 |
| 高价股归属 | 被推向>Q75档 | 分布更均匀 |
| 跨股票可比性 | 差（受股价影响大） | 好（已消除价格因素） |
| 各档样本量 | 受股价结构性影响 | 更均匀地反映"相对幅度" |

### 1.4 数据范围

与原报告一致：2016-01-01 ~ 2026-03-18（近10年），衰减权重半衰期3.5年。

| 层级 | 样本量 |
|------|--------|
| 大盘指数 | 7大指数，日线 峰295+谷296, 周线 峰114+谷116 |
| 行业指数 | 31个申万一级行业（5大类别），日线 峰1289+谷1319, 周线 峰514+谷539 |
| 个股 | ~5450只A股，日线 峰171,231+谷173,000, 周线 峰66,196+谷69,005, 月线 峰22,079+谷24,264 |

## 六、结论：是否应将ATR标准化纳入信号系统

### 6.1 ATR标准化的价值判定

| 评估项 | 结论 |
|--------|------|
| **是否消除了偏差？** | 是。DIF/ATR在不同股价、不同行业间分布高度一致（中位数0.77~0.92） |
| **幅度梯度是否更清晰？** | 是。峰值T+20梯度从10.6pp扩大到15.8pp |
| **是否修正了错误结论？** | 是。原报告"大幅度长期更强"的结论被推翻，那是高价股效应 |
| **是否改变了核心结论？** | 不改变。"小幅度短中期更优"的规律更加显著 |
| **信号衰减曲线是否变化？** | 不变。全量胜率与分层方式无关 |

### 6.2 建议

**应将DIF/ATR纳入信号系统，取代原始|DIF|作为幅度标注指标。** 具体建议：

1. **信号输出改用DIF/ATR**：极值信号中附加 DIF/ATR 值（而非 |DIF|），使跨股票可比
2. **分档标准**：可使用全市场统一的DIF/ATR分位数（而非每只股票单独计算），提供统一的"相对强度"度量
3. **废弃"大DIF长期更强"的结论**：这是价格偏差导致的伪发现，标准化后不成立
4. **保留核心策略不变**：
   - 小幅度极值（DIF/ATR < Q25）→ 短中期信号最纯净，适合T+5~T+20操作
   - 大幅度极值（DIF/ATR > Q75）→ 短期胜率仍然很高（峰值T+5=92.4%，谷值T+5=88.1%），但梯度上逊于小幅度
   - 月线信号仍是最高价值信号（T+3m胜率95%+），与幅度分层方式无关
5. **实际部署阈值参考**：DIF/ATR ≈ 1.0 对应典型的DIF极值（中位数附近），DIF/ATR > 1.3（>Q75）代表相对波动率而言的强极值
