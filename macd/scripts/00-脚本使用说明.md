# MACD指标研究 - 脚本说明

## 计算脚本

| 脚本 | 用途 |
|------|------|
| `run_all.py` | 一键执行全部MACD计算（建表 + 股票 + 指数） |
| `compute_index_macd.py` | 指数MACD全市场计算入口 |
| `compute_stock_macd.py` | 股票MACD全市场计算入口 |
| `compute_stock_macd_signals.py` | 个股MACD信号事件回填，检测交叉/零轴穿越/背离/DIF极值并计算后续收益 |

## 分析脚本

| 脚本 | 用途 |
|------|------|
| `analyze_index_macd.py` | Layer 1: 上证指数MACD全维度分析（信号统计 + 收益 + 牛熊交叉 + 顶底特征） |
| `analyze_stock_macd.py` | Layer 4: 个股MACD信号统计分析，汇总写入 stock_macd_signal_stats 表 |
| `analyze_sw_macd.py` | Layer 3: 申万一级行业MACD信号分析，验证指数结论的行业适用性 |
| `analyze_three_points.py` | MACD三个关键位置分析：DIF极大值/极小值/零轴，三指数对比 |
| `analyze_dif_proximity.py` | DIF极值接近度分析，统计各接近度区间的未来收益和胜率 |
| `analyze_dif_extreme_effectiveness.py` | DIF极值信号有效性深度分析：幅度阈值×信号质量、有效窗口衰减、三层级（大盘/行业/个股） |
| `analyze_dif_extreme_atr.py` | DIF极值有效性分析（ATR标准化版），用 DIF/ATR(20) 替代原始\|DIF\|做幅度分层 |

## 基础模块

| 脚本 | 用途 |
|------|------|
| `config.py` | 数据库连接配置（读 my_stock / 写 stock_research） |
| `database.py` | 双引擎初始化（read_engine + write_engine） |
| `db_utils.py` | batch_upsert 批量写入工具 |
| `kline_loader.py` | K线数据加载 + 复权（从 my_stock 库读取） |
| `macd_calc.py` | MACD计算核心逻辑（EMA + DIF/DEA/MACD） |
| `models.py` | 16张MACD表模型定义（股票12张 + 指数4张） |
| `signal_detector.py` | MACD信号检测引擎（金叉/死叉/背离/零轴穿越/DIF极值） |
| `bull_bear_phases.py` | A股牛熊周期数据 + 标注工具 |
